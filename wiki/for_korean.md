# Go 언어의 "for" 루프: 사용법과 예제

## 개요
Go 프로그래밍 언어에서 "for" 루프는 반복문을 구현하는 데 사용됩니다. 이는 Go의 유일한 반복문으로, 다양한 형태로 사용 가능하여 코드의 흐름을 제어하는 데 중요한 역할을 합니다.

## 문서
"for" 루프는 반복 작업을 수행하는 간단한 구문을 제공합니다. Go에서 "for"는 다음과 같은 세 가지 형태로 사용할 수 있습니다:

1. **기본 형태**: 초기화, 조건, 후속 작업을 포함하는 전통적인 for 루프.
   ```go
   for i := 0; i < 10; i++ {
       fmt.Println(i)
   }
   ```

2. **조건만 있는 형태**: 초기화 및 후속 작업 없이 조건만 있는 루프.
   ```go
   i := 0
   for i < 10 {
       fmt.Println(i)
       i++
   }
   ```

3. **무한 루프**: 조건을 생략하여 무한히 반복하는 루프.
   ```go
   for {
       fmt.Println("무한 루프")
   }
   ```

이러한 형태를 통해 다양한 반복 작업을 쉽게 수행할 수 있습니다.

## 예제
기본적인 "for" 루프 사용 예제:
```go
package main

import "fmt"

func main() {
    // 기본 for 루프
    for i := 0; i < 5; i++ {
        fmt.Println("인덱스:", i)
    }

    // 조건만 있는 for 루프
    j := 0
    for j < 5 {
        fmt.Println("인덱스:", j)
        j++
    }

    // 무한 루프
    k := 0
    for {
        if k >= 5 {
            break
        }
        fmt.Println("인덱스:", k)
        k++
    }
}
```

## 설명
Go의 "for" 루프는 간단하지만 강력한 도구입니다. 그러나 몇 가지 주의할 점이 있습니다:

- **무한 루프**: 조건을 생략한 경우, 종료 조건을 명시적으로 설정하지 않으면 프로그램이 무한히 실행될 수 있습니다. 이는 시스템 자원을 소모하므로 주의해야 합니다.
- **변수 Scope**: 루프 내에서 선언된 변수는 루프가 종료된 후에도 접근할 수 없습니다. 따라서 루프 외부에서 변수를 사용하려면 사전에 선언해야 합니다.
- **중첩 루프**: "for" 루프는 중첩될 수 있습니다. 그러나 이 경우 성능을 고려해야 하며, 가독성이 떨어질 수 있습니다.

## 한 줄 요약
Go 언어의 "for" 루프는 다양한 반복 작업을 간단하게 처리할 수 있는 유일한 반복문입니다.
