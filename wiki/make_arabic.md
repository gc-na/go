# استخدام الأمر "make" في لغة Go: دليل شامل

## ملخص
الأمر "make" في لغة Go هو دالة مدمجة تُستخدم لإنشاء وتخصيص الشرائح (slices) والقوائم (maps) والقوائم المرتبطة (channels). يوفر الأمر "make" وسيلة فعالة لتخصيص الذاكرة وتحديد السعة الأولية للهياكل المعقدة.

## الوثائق
### الغرض
تستخدم دالة "make" في Go لإنشاء هياكل بيانات ديناميكية مثل القوائم الشرائحية (slices) والقوائم (maps) والقنوات (channels). هذه الهياكل تسمح بتخزين البيانات وإدارتها بكفاءة.

### الاستخدام
تُستخدم دالة "make" بالشكل التالي:

```go
make(t نوع, السعة الأولية ...int) نوع
```

- **t**: النوع المراد إنشاؤه (مثل `slice` أو `map` أو `channel`).
- **السعة الأولية**: عدد العناصر التي يمكن أن يحتوي عليها الهيكل عند إنشائه، وهذا اختياري.

### التفاصيل
- **القوائم الشرائحية (Slices)**: تُستخدم لإنشاء شريحة جديدة. إذا تم تحديد السعة الأولية، سيتم تخصيص الذاكرة لذلك العدد من العناصر.
- **القوائم (Maps)**: تُستخدم لإنشاء خريطة جديدة. يمكن تقديم سعة أولية لزيادة كفاءة تخصيص الذاكرة.
- **القنوات (Channels)**: تُستخدم لإنشاء قناة جديدة. السعة الأولية تحدد عدد الرسائل التي يمكن أن تحملها القناة قبل أن تُغلق.

## أمثلة
### مثال على إنشاء شريحة
```go
package main

import "fmt"

func main() {
    slice := make([]int, 0, 5) // إنشاء شريحة بسعة أولية 5
    fmt.Println(slice) // []
}
```

### مثال على إنشاء خريطة
```go
package main

import "fmt"

func main() {
    m := make(map[string]int) // إنشاء خريطة جديدة
    m["one"] = 1
    m["two"] = 2
    fmt.Println(m) // map[one:1 two:2]
}
```

### مثال على إنشاء قناة
```go
package main

import "fmt"

func main() {
    ch := make(chan int, 2) // إنشاء قناة بسعة أولية 2
    ch <- 1
    ch <- 2
    fmt.Println(<-ch) // 1
}
```

## الشرح
عند استخدام "make"، يجب أن تكون على دراية ببعض النقاط المهمة:
- إذا كنت تستخدم "make" مع الشرائح، فإنها لا تُنشئ مصفوفة ثابتة، بل تُنشئ شريحة ديناميكية يمكن أن تنمو.
- لا يمكن استخدام "make" لإنشاء أنواع مثل الهياكل (structs) أو المصفوفات الثابتة (arrays) - بل تُستخدم لإنشاء هياكل بيانات ديناميكية فقط.
- السعة الأولية ليست إلزامية، ولكن استخدامها يمكن أن يحسن الأداء عن طريق تقليل الحاجة لإعادة تخصيص الذاكرة.

## ملخص بجملة واحدة
يُعد الأمر "make" في Go أداة أساسية لإنشاء وتخصيص القوائم الشرائحية والقوائم والقنوات الديناميكية بكفاءة.
