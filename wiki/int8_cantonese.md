# int8 在 Go 語言中的用法與特性

## 概述
`int8` 是 Go 語言中的一種整數數據類型，用於表示範圍從 -128 到 127 的有符號整數。由於其佔用 1 個字節的內存，`int8` 通常用於需要節省內存的情況。

## 文檔
`int8` 是 Go 語言中的基本數據類型之一，主要用於處理小範圍的整數值。使用 `int8` 的好處在於它的內存占用較小，適合在大型數據結構中使用，例如數組或切片，從而提高性能和減少存儲需求。

### 用法
在 Go 語言中，`int8` 可以通過以下方式聲明：

```go
var num int8
```

也可以在聲明時給予初始值：

```go
var num int8 = 100
```

### 特性
- **範圍**: `int8` 的取值範圍是 -128 到 127。
- **內存占用**: 每個 `int8` 變量占用 1 個字節。
- **運算**: `int8` 支持所有基本的算術運算，如加法、減法、乘法和除法。
- **類型轉換**: 在與其他整數類型（如 `int`, `int16`, `int32` 等）運算時，`int8` 會自動轉換為更大的整數類型。

## 範例
以下是使用 `int8` 的基本範例：

```go
package main

import "fmt"

func main() {
    var num1 int8 = 50
    var num2 int8 = 75
    var sum int8 = num1 + num2

    fmt.Println("Sum:", sum) // 輸出: Sum: 125

    var num3 int8 = 127
    // 注意: 對於超過 int8 最大值的運算
    num3 += 1 // 會導致溢出，變成 -128
    fmt.Println("Overflowed num3:", num3) // 輸出: Overflowed num3: -128
}
```

## 解釋
使用 `int8` 時需要注意以下幾點：
- **溢出**: 如果計算結果超過 `int8` 的範圍，將發生溢出，這會導致結果變成負數或意外的值。
- **類型轉換**: 在與其他數據類型進行運算時，應該明確指定類型轉換，以避免不必要的錯誤。
- **性能**: 儘管 `int8` 節省內存，但在某些情況下，使用更大的整數類型（如 `int` 或 `int32`）可能會更高效，特別是在需要進行大量數學運算時。

## 一句總結
`int8` 是 Go 語言中一種有用的整數數據類型，適合用於小範圍整數的存儲和處理，但需注意溢出問題。
