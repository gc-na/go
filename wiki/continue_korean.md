# Go 언어의 continue 문법: 반복문에서의 제어 흐름 관리

## 개요
Go 언어의 `continue` 문은 반복문에서 특정 조건을 만족할 때 현재 반복을 건너뛰고 다음 반복으로 넘어갈 수 있게 해주는 제어 흐름 명령어입니다. 이를 통해 코드의 가독성을 높이고, 불필요한 연산을 줄일 수 있습니다.

## 문서화

### 목적
`continue` 문은 반복문 내에서 특정 조건을 만족하는 경우, 해당 반복의 나머지 부분을 실행하지 않고 다음 반복으로 넘어가도록 합니다. 이는 주로 `for` 루프와 함께 사용되며, 조건이 충족될 때 불필요한 코드를 실행하지 않도록 도와줍니다.

### 사용법
`continue` 문은 다음과 같은 형식으로 사용됩니다:

```go
for 조건 {
    if 특정조건 {
        continue // 다음 반복으로 넘어감
    }
    // 반복할 코드
}
```

- `조건`: 반복문이 실행되는 동안 참일 경우 계속 반복됩니다.
- `특정조건`: 이 조건이 참일 경우, `continue` 문이 실행되어 현재 반복을 건너뜁니다.

### 세부사항
- `continue` 문은 중첩된 반복문에서도 사용할 수 있으며, 가장 가까운 반복문에만 적용됩니다.
- `continue` 문은 `switch` 문과 함께 사용할 수 없으며, 오직 반복문에서만 사용해야 합니다.

## 예시

### 기본 사용 예제
다음은 `continue` 문을 사용하는 간단한 예제입니다. 이 코드는 1부터 10까지의 수 중에서 짝수만 출력합니다.

```go
package main

import "fmt"

func main() {
    for i := 1; i <= 10; i++ {
        if i%2 != 0 {
            continue // 홀수는 건너뜁니다
        }
        fmt.Println(i) // 짝수만 출력
    }
}
```

### 중첩된 반복문 예제
다음은 중첩된 반복문에서 `continue` 문을 사용하는 예입니다.

```go
package main

import "fmt"

func main() {
    for i := 1; i <= 3; i++ {
        for j := 1; j <= 3; j++ {
            if j == 2 {
                continue // j가 2일 때 다음 j 반복으로 넘어감
            }
            fmt.Printf("i: %d, j: %d\n", i, j)
        }
    }
}
```

## 설명
`continue` 문을 사용할 때 주의할 점은 조건이 잘못 설정되면 원하지 않는 결과를 초래할 수 있다는 것입니다. 예를 들어, 특정 조건이 항상 참일 경우 무한 루프에 빠질 수 있습니다. 또한, `continue` 문을 사용할 때 반복문 내의 다른 코드가 무시되므로, 그 코드가 반드시 실행되어야 하는 경우에는 적절히 조건을 설정해야 합니다.
