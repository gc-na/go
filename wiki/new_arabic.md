# استخدام الكلمة المفتاحية "new" في لغة Go: إنشاء مثيلات جديدة من الكائنات

## ملخص
تُستخدم الكلمة المفتاحية "new" في لغة Go لإنشاء مثيلات جديدة من الكائنات. يُعتبر "new" أداة قوية تسمح للمطورين بإنشاء كائنات من أنواع مختلفة وتخصيص الذاكرة لها بطريقة مباشرة.

## الوثائق
تُستخدم الكلمة المفتاحية "new" في Go لإنشاء كائن جديد من نوع معين. تقوم هذه الكلمة بإرجاع مؤشر إلى الكائن الذي تم إنشاؤه. يتم تخصيص الذاكرة للكائن في وقت التشغيل، مما يسمح للمطورين بإنشاء أنواع مختلفة من الكائنات بسهولة.

### الاستخدام
يمكن استخدام "new" مع أي نوع بيانات، سواء كانت بنى (structs) أو أنواع بيانات بسيطة. على سبيل المثال:

```go
p := new(Person)
```

في هذا المثال، يتم إنشاء كائن جديد من النوع `Person`، ويتم إرجاع مؤشر إلى هذا الكائن.

### التفاصيل
- يقوم `new` بتهيئة الذاكرة بما يتناسب مع حجم النوع المعني.
- يتم تعيين القيم الافتراضية للحقول، مثل 0 للأعداد و nil للسلاسل.
- النوع المعاد هو مؤشر (pointer) إلى الكائن، مما يعني أنه يمكن استخدامه لتعديل الكائن مباشرة.

## الأمثلة
### مثال 1: إنشاء كائن من هيكل
```go
type Person struct {
    Name string
    Age  int
}

func main() {
    p := new(Person)
    p.Name = "أحمد"
    p.Age = 30
    fmt.Println(p)
}
```

### مثال 2: استخدام new مع الأنواع المدمجة
```go
func main() {
    x := new(int)
    *x = 42
    fmt.Println(*x)
}
```

## الشرح
رغم أن "new" بسيطة في استخدامها، هناك بعض الأمور التي يجب الانتباه إليها:

- **المؤشرات**: `new` يعود بمؤشر، لذا يجب عليك استخدام النجمة (`*`) للوصول إلى القيم.
- **التهيئة**: يتم تعيين القيم الافتراضية، لذا تأكد من تهيئة الحقول إذا لزم الأمر.
- **التخصيص**: إذا كنت بحاجة إلى تخصيص الكائن بشكل خاص، قد يكون من الأفضل استخدام الدوال المُنشئة (constructor functions) بدلاً من `new`.

## ملخص جملة واحدة
تُستخدم "new" في Go لإنشاء كائنات جديدة من الأنواع المختلفة مع تخصيص الذاكرة وإرجاع مؤشر إليها.
