# Go 語言中的整數型別 (int)

## 簡介
在 Go 語言中，`int` 是一種基本的整數型別，用於表示整數數值。根據平台的不同，`int` 型別的大小可能為 32 位或 64 位，這使得它在性能和內存使用方面非常高效。

## 文檔
### 目的
`int` 型別是 Go 語言中最常用的整數型別之一，適合用於各種算法和數據處理的場合。由於它的大小依賴於運行環境，`int` 型別是編寫可移植代碼的理想選擇。

### 用法
在 Go 中，`int` 型別可以直接使用數字來定義，並可用於數學運算、循環控制以及數據結構中的索引等。

### 詳細說明
- **定義**：使用變量宣告來創建 `int` 型別的變量。
- **範圍**：`int` 的範圍取決於架構，通常在 32 位系統上範圍為 -2,147,483,648 到 2,147,483,647，在 64 位系統上範圍為 -9,223,372,036,854,775,808 到 9,223,372,036,854,775,807。
- **自動類型轉換**：在某些情況下，`int` 型別會自動轉換為其他數字型別，但在不同型別之間進行運算時需注意類型不匹配的情況。
- **零值**：`int` 型別的零值為 0，未初始化的 `int` 變量將默認為 0。

## 範例
以下是使用 `int` 型別的基本範例：

```go
package main

import (
    "fmt"
)

func main() {
    var a int = 10
    var b int = 20
    sum := a + b
    fmt.Println("兩數之和為:", sum) // 輸出: 兩數之和為: 30
}
```

## 解釋
- **常見陷阱**：使用 `int` 時，需要注意整數溢出的情況。例如，對於大型整數計算時，可能會超出 `int` 的範圍，導致錯誤的計算結果。
- **類型轉換**：在進行數學運算時，確保所有參與運算的變量均為相同的數字型別，以避免類型不匹配的錯誤。
- **性能考量**：`int` 型別通常比其他整數型別（如 `int8`、`int16`、`int32`）在性能上更優，特別是在需要處理大量數據時。

## 一句總結
`int` 是 Go 語言中用來表示整數的基本型別，具有高效的性能和可移植性。
