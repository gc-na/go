# Go 언어의 맵(Map): 데이터 구조의 이해와 활용

## 개요
Go 언어에서 맵(Map)은 키-값 쌍을 저장하는 비순차적 데이터 구조입니다. 효율적인 데이터 검색 및 관리를 가능하게 하며, 동적 크기 조정이 가능합니다.

## 문서화
### 목적
맵은 해시 테이블을 기반으로 하여, 특정 키에 대한 값을 빠르게 조회할 수 있도록 설계되었습니다. 이는 데이터베이스와 같은 구조에서 자주 사용되며, 키를 통해 값에 즉시 접근할 수 있는 장점을 제공합니다.

### 사용법
Go에서 맵을 선언하고 사용하는 방법은 다음과 같습니다:

1. **맵 선언**: `make` 함수를 사용하거나 리터럴을 사용하여 맵을 생성할 수 있습니다.
   ```go
   // make 함수 사용
   myMap := make(map[string]int)

   // 리터럴 사용
   anotherMap := map[string]int{"a": 1, "b": 2}
   ```

2. **값 추가 및 수정**:
   ```go
   myMap["key1"] = 100  // 값 추가
   myMap["key1"] = 200  // 값 수정
   ```

3. **값 조회**:
   ```go
   value := myMap["key1"]  // 값을 조회
   ```

4. **값 삭제**:
   ```go
   delete(myMap, "key1")  // 값 삭제
   ```

5. **키 존재 여부 확인**:
   ```go
   value, exists := myMap["key1"]  // 존재 여부 확인
   ```

### 세부사항
- 맵의 키는 고유해야 하며, 같은 키에 대해 여러 값을 저장할 수 없습니다.
- 맵은 동적 크기 조정이 가능하므로, 필요에 따라 자동으로 크기가 조정됩니다.
- 맵의 키와 값은 모든 타입이 될 수 있지만, 키는 반드시 해시 가능해야 합니다.

## 예제
```go
package main

import "fmt"

func main() {
    // 맵 생성
    ages := make(map[string]int)

    // 값 추가
    ages["Alice"] = 30
    ages["Bob"] = 25

    // 값 수정
    ages["Alice"] = 31

    // 값 조회
    fmt.Println("Alice의 나이:", ages["Alice"])

    // 값 삭제
    delete(ages, "Bob")

    // 키 존재 여부 확인
    if age, exists := ages["Bob"]; exists {
        fmt.Println("Bob의 나이:", age)
    } else {
        fmt.Println("Bob의 정보가 없습니다.")
    }
}
```

## 설명
- **공통적인 함정**: 맵의 값을 조회할 때, 키가 존재하지 않는 경우 기본값을 반환합니다. 예를 들어, `int` 타입의 기본값은 0이므로, 키가 존재하지 않는데 값을 0으로 간주할 수 있습니다.
- **맵의 동시성**: Go의 맵은 동시 읽기 및 쓰기를 지원하지 않으므로, 여러 고루틴이 동일한 맵에 접근할 경우 `sync.Mutex`와 같은 동기화 메커니즘을 사용해야 합니다.
