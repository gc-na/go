# Go 프로그래밍 언어의 int8: 타입과 활용법

## 개요
Go 언어에서 `int8`은 8비트 정수 타입으로, -128에서 127까지의 값을 표현할 수 있습니다. 이 타입은 메모리 사용을 최적화하고, 특정 범위의 정수를 다루는 데 유용합니다.

## 문서화
`int8`은 Go 언어의 기본 데이터 타입 중 하나로, signed integer(부호 있는 정수)입니다. `int8` 타입의 변수는 1바이트의 메모리를 차지하며, 주로 메모리를 절약하고 성능을 높이기 위해 사용됩니다.

### 목적
- 메모리 효율성을 높이는 데 도움을 줍니다.
- 특정 범위의 정수 값을 다룰 때 사용됩니다.

### 사용법
`int8`은 변수 선언 시 사용되며, 기본적인 연산과 함께 활용할 수 있습니다. 다음은 `int8` 타입 변수를 선언하고 사용하는 기본적인 방법입니다.

```go
var a int8 = 100
var b int8 = -50
var c int8 = a + b
```

이 예제에서 `c`는 50이 됩니다.

## 예제
다음은 `int8`을 사용하는 간단한 예제입니다.

```go
package main

import "fmt"

func main() {
    var num1 int8 = 10
    var num2 int8 = 20
    
    sum := num1 + num2
    fmt.Println("합계:", sum) // 출력: 합계: 30

    var overflow int8 = 127 + 1 // overflow 발생
    fmt.Println("오버플로우 결과:", overflow) // 출력: 오버플로우 결과: -128
}
```

이 코드에서는 `int8` 타입의 숫자를 더하는 예제를 보여주며, 오버플로우 발생 시의 결과도 확인할 수 있습니다.

## 설명
`int8`을 사용할 때 주의해야 할 점은 오버플로우입니다. `int8`의 최대값은 127이며, 이를 초과하는 값은 다시 -128로 돌아갑니다. 따라서, 프로그램에서 `int8` 타입 변수를 사용할 때는 이러한 범위를 항상 고려해야 합니다. 

또한, 다른 정수 타입과의 연산 시에는 타입 변환이 필요할 수 있으며, 이를 무시하면 컴파일 오류가 발생할 수 있습니다.

## 한 줄 요약
Go 언어의 `int8`은 -128에서 127까지의 값을 저장할 수 있는 8비트 부호 있는 정수 타입입니다.
