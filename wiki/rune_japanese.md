# Go言語における「rune」の完全ガイド

## 概要
Go言語における「rune」は、Unicode文字を表すためのデータ型です。文字列内の個々の文字を効率的に処理するために使用されます。

## ドキュメンテーション
### 目的
`rune`は、Go言語において、Unicodeコードポイントを表すためのエイリアスであり、整数型`int32`の別名です。これにより、プログラマーはUTF-8エンコーディングで表現された文字を扱うことができます。

### 使用法
`rune`は、文字列から文字を取得したり、文字を文字列に変換したりする際に利用されます。以下は、`rune`を使った基本的な操作の例です。

### 詳細
- `rune`は、Unicodeの各コードポイントに対応するため、国際化されたアプリケーションの開発において非常に重要です。
- `rune`は、Go言語の標準ライブラリ内の多くの関数で使用されており、特に文字列処理やテキスト操作に役立ちます。

## 例
以下は、Go言語で`rune`を使用する基本的な例です。

```go
package main

import (
    "fmt"
)

func main() {
    // 文字列をruneスライスに変換
    str := "こんにちは"
    runes := []rune(str)

    // 各runeを表示
    for i, r := range runes {
        fmt.Printf("Index: %d, Rune: %c, Unicode: %U\n", i, r, r)
    }
}
```

このコードは、文字列「こんにちは」を`rune`スライスに変換し、各文字のインデックス、文字自体、そしてUnicodeコードポイントを表示します。

## 説明
- **一般的な落とし穴**: `rune`は`byte`とは異なり、1つの`rune`が必ずしも1バイトであるとは限りません。UTF-8エンコーディングでは、特定の文字は複数のバイトで表されるため、`len`関数を使用して文字列のバイト数を取得すると、`rune`の数とは異なる結果になることがあります。
- **注意点**: 文字列を`rune`に変換する際には、文字列がUTF-8でエンコードされていることを確認してください。無効なUTF-8シーケンスが含まれている場合、変換は失敗します。

## 一文の要約
Go言語の`rune`は、Unicode文字を効率的に扱うためのデータ型であり、国際化対応のアプリケーション開発に不可欠です。
