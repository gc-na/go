# Go 語言中的 const：常數的使用與最佳實踐

## 簡介
在 Go 語言中，`const` 關鍵字用於聲明常數。常數是在程序執行期間不會改變的值，這使得它們在編碼中非常有用。使用常數可以提高代碼的可讀性和可維護性。

## 文檔
### 目的
`const` 使開發者能夠定義固定的值，這些值在編譯期間就已確定，並且在整個程序中使用時不會發生變化。常數可以是任何基本數據類型，包括整數、浮點數、布爾值和字符串。

### 使用方法
在 Go 中，使用 `const` 關鍵字來聲明常數，語法如下：

```go
const 常數名稱 型別 = 值
```

你也可以省略型別，讓編譯器自動推斷：

```go
const 常數名稱 = 值
```

常數的範圍可以是包級別，也可以是函式級別。常數可以在多個地方使用，並且不會占用運行時的內存。

### 詳細說明
- 常數在編譯期間就會被替換為其值，因此不會有運行時的性能開銷。
- 常數不支援變數的賦值運算，比如 `+=` 或 `++`。
- 常數可以使用 `iota` 來生成一組連續的數值，這在定義枚舉時非常有用。

## 範例
以下是一些使用 `const` 的基本示例：

```go
package main

import "fmt"

const Pi = 3.14
const Greeting = "Hello, World!"

func main() {
    fmt.Println("Pi 的值是:", Pi)
    fmt.Println(Greeting)
}
```

使用 `iota` 的示例：

```go
package main

import "fmt"

const (
    A = iota // 0
    B        // 1
    C        // 2
)

func main() {
    fmt.Println("A:", A, "B:", B, "C:", C)
}
```

## 解釋
- **常見陷阱**：當你聲明常數時，確保不會對其進行修改，因為常數一旦定義就不可更改。
- **型別推斷**：如果不指定型別，常數的型別會根據值自動推斷，但在某些情況下可能會導致預期外的行為，特別是在數學運算中。
- **作用範圍**：包級別的常數可以在整個包中使用，而函式級別的常數只能在該函式內部使用。

## 總結
`const` 是 Go 語言中用於聲明不可變的常數的重要工具，能夠提升代碼的可讀性和性能。
