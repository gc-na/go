# طباعة في لغة Go: الدليل الشامل

## الملخص
تعتبر دالة الطباعة من أهم الأدوات في لغة Go، حيث تتيح للمطورين عرض البيانات والمعلومات على الشاشة، مما يسهل عملية تصحيح الأخطاء وفهم سلوك البرنامج.

## الوثائق
تستخدم دالة `fmt.Print` ودالة `fmt.Println` في حزمة `fmt` لطباعة النصوص والبيانات في لغة Go. الغرض الرئيسي من هذه الدوال هو عرض المعلومات على وحدة التحكم. 

### الاستخدام
- **`fmt.Print`**: تطبع البيانات دون إضافة سطر جديد في النهاية.
- **`fmt.Println`**: تطبع البيانات مع إضافة سطر جديد في النهاية.
- **`fmt.Printf`**: تطبع البيانات باستخدام تنسيق محدد يمكن تحديده من خلال سلسلة تنسيق.

### تفاصيل
- **المتغيرات**: يمكن استخدام أي نوع من البيانات (مثل الأعداد، النصوص، أو الهياكل) مع دوال الطباعة.
- **التنسيق**: باستخدام `fmt.Printf`، يمكنك التحكم في شكل البيانات المعروضة باستخدام رموز تنسيق مثل `%s` للنصوص و`%d` للأعداد.

## الأمثلة
### مثال 1: استخدام `fmt.Print`
```go
package main

import "fmt"

func main() {
    fmt.Print("مرحبًا بالعالم!") // ستظهر بدون سطر جديد
}
```

### مثال 2: استخدام `fmt.Println`
```go
package main

import "fmt"

func main() {
    fmt.Println("مرحبًا بالعالم!") // ستظهر مع سطر جديد
}
```

### مثال 3: استخدام `fmt.Printf`
```go
package main

import "fmt"

func main() {
    name := "أحمد"
    age := 30
    fmt.Printf("اسمي %s وعمري %d سنة.\n", name, age) // تنسيق مخصص
}
```

## الشرح
### الأخطاء الشائعة
- **نسيان استيراد حزمة `fmt`**: تأكد من استيراد الحزمة قبل استخدامها.
- **استخدام التنسيق الخاطئ**: عند استخدام `fmt.Printf`، تأكد من أن عدد رموز التنسيق يتطابق مع عدد المتغيرات المُعطاة.
- **التعامل مع نوع البيانات غير الصحيح**: تأكد من استخدام نوع البيانات المناسب لكل رمز تنسيق.

## ملخص بعبارة واحدة
تعتبر دوال الطباعة في Go أدوات أساسية لعرض البيانات، مما يساعد المطورين على تصحيح الأخطاء وفهم البرنامج.
