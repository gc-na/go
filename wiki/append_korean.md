# Go 프로그래밍 언어의 append 함수: 사용법과 예제

## 개요
Go 언어의 `append` 함수는 슬라이스에 요소를 추가하는 데 사용됩니다. 이 함수는 슬라이스의 길이를 동적으로 조절할 수 있어 데이터 구조를 유연하게 관리할 수 있게 해줍니다.

## 문서화

### 목적
`append` 함수는 기존 슬라이스에 새로운 요소를 추가하고, 필요할 경우 새로운 슬라이스를 반환합니다. 이는 Go 언어의 슬라이스가 고정 크기가 아닌 동적 크기를 가질 수 있도록 돕습니다.

### 사용법
`append` 함수의 기본 형식은 다음과 같습니다:

```go
func append(slice []Type, elements ...Type) []Type
```

- `slice`: 요소를 추가할 기존 슬라이스입니다.
- `elements`: 추가할 요소들입니다. 가변 인수를 사용하여 여러 개의 요소를 동시에 추가할 수 있습니다.

이 함수는 새로운 슬라이스를 반환하며, 원래의 슬라이스는 변경되지 않을 수 있습니다. 만약 원래 슬라이스의 용량이 충분하지 않다면, 새로운 슬라이스가 할당됩니다.

### 세부 사항
- `append` 함수는 슬라이스가 가득 찼을 때 자동으로 용량을 증가시킵니다.
- 반환된 슬라이스는 원래 슬라이스와 다를 수 있으며, 원래 슬라이스를 계속 사용할 수 있습니다.
- 슬라이스의 길이와 용량을 이해하는 것이 중요합니다. 길이는 슬라이스의 실제 데이터 개수를 나타내고, 용량은 할당된 최대 크기를 나타냅니다.

## 예제

### 기본 사용법
```go
package main

import "fmt"

func main() {
    slice := []int{1, 2, 3}
    slice = append(slice, 4, 5)
    fmt.Println(slice) // 출력: [1 2 3 4 5]
}
```

### 여러 요소 추가
```go
package main

import "fmt"

func main() {
    slice := []string{"a", "b", "c"}
    slice = append(slice, "d", "e", "f")
    fmt.Println(slice) // 출력: [a b c d e f]
}
```

### 슬라이스의 슬라이스
```go
package main

import "fmt"

func main() {
    slice1 := []int{1, 2, 3}
    slice2 := []int{4, 5}
    slice1 = append(slice1, slice2...) // 슬라이스를 펼쳐서 추가
    fmt.Println(slice1) // 출력: [1 2 3 4 5]
}
```

## 설명
- **슬라이스의 용량 초과**: 슬라이스의 용량을 초과하여 요소를 추가하면, `append`는 자동으로 새로운 슬라이스를 생성합니다. 이로 인해 원래 슬라이스는 영향을 받지 않지만, 새로운 슬라이스를 사용해야 합니다.
- **슬라이스의 복사**: 슬라이스를 다른 슬라이스에 할당하면, 원래 슬라이스와 복사된 슬라이스는 같은 데이터를 참조합니다. 이를 통해 한 슬라이스의 변경이 다른 슬라이스에 영향을 줄 수 있습니다.
- **불변성 주의**: `append`의 결과를 원래 슬라이스에 다시 할당해야 합니다. 그렇지 않으면 업데이트가 반영되지 않습니다.

## 한 줄 요약
Go 언어의 `append` 함수는 슬라이스에 요소를 추가하고, 필요 시 새로운 슬라이스를 생성하여 동적 데이터를 관리할 수 있게 해준다.
