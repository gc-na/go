# Go 語言中的常量 (const) 使用指南

## 概述
在 Go 語言中，`const` 關鍵字用來定義常量，這些常量在程式運行過程中不會改變。使用常量可以提高程式的可讀性和維護性，並減少硬編碼的錯誤。

## 文檔
`const` 是 Go 語言的一個關鍵字，用於聲明常量。常量可以是數字、字串或布林值。常量在初始化時被賦值，並且在程式的生命週期內保持不變。使用常量的好處包括：

- **提高可讀性**：通過命名常量，可以使程式碼更易於理解。
- **類型安全**：常量的類型在編譯時檢查，減少了運行時錯誤的可能性。
- **性能優勢**：常量在編譯時被處理，運行時不需要額外的資源。

### 使用語法
```go
const 常量名稱 常量類型 = 常量值
```
或者不指定類型，Go 將自動推斷：
```go
const 常量名稱 = 常量值
```

## 範例
以下是一些使用 `const` 的基本範例：

```go
package main

import "fmt"

const Pi = 3.14
const Greeting = "你好，世界！"
const IsActive = true

func main() {
    fmt.Println("圓周率是:", Pi)
    fmt.Println(Greeting)
    fmt.Println("是否啟用:", IsActive)
}
```

### 例子說明
在這個範例中，我們定義了三個常量：`Pi`、`Greeting` 和 `IsActive`。這些常量在程式執行過程中不會改變，並且可以直接使用。

## 解釋
使用 `const` 時需要注意以下幾點：

1. **類型推斷**：如果不提供類型，Go 將自動推斷常量的類型，但這可能會導致某些操作不符合預期。
2. **不可變性**：常量在運行時不允許改變，這可能會導致錯誤的假設，例如試圖在函數內重新賦值常量。
3. **常量表達式**：常量可以是常量表達式的結果，例如：
   ```go
   const (
       A = 1
       B = A + 2  // B 的值為 3
   )
   ```

## 一句總結
在 Go 語言中，`const` 用來定義不變的常量，有助於提高程式碼的可讀性和安全性。
