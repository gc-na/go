# Go 언어의 "if" 문: 조건문 사용법

## 개요
Go 언어에서의 "if" 문은 조건에 따라 프로그램의 흐름을 제어하는 데 사용되는 핵심적인 제어 구조입니다. 조건이 참일 경우 특정 코드 블록을 실행하고, 그렇지 않은 경우 다른 코드 블록을 실행하는 데 유용합니다.

## 문서화

### 목적
"if" 문은 조건을 평가하고 그 결과에 따라 실행할 코드 블록을 결정하는 데 사용됩니다. 이는 프로그램의 논리를 구성하는 데 필수적인 요소입니다.

### 사용법
Go에서 "if" 문은 다음과 같은 구조를 가집니다:

```go
if 조건 {
    // 조건이 참일 때 실행할 코드
} else {
    // 조건이 거짓일 때 실행할 코드
}
```

* 조건: 불리언 값을 반환하는 표현식.
* 중괄호 `{}`: 블록을 감싸는 데 사용되며, 조건이 참일 때 실행될 코드가 포함됩니다.

### 상세 정보
- **else if**: 여러 조건을 검사할 때 사용할 수 있습니다.
  
```go
if 조건1 {
    // 조건1이 참일 때 실행
} else if 조건2 {
    // 조건2가 참일 때 실행
} else {
    // 위의 모든 조건이 거짓일 때 실행
}
```

- **초기화 문**: "if" 문 내에서 변수를 선언하고 초기화할 수 있습니다. 이 변수는 "if" 블록 내에서만 유효합니다.

```go
if x := 10; x > 5 {
    // x가 5보다 클 때 실행
}
```

## 예제

### 기본 사용 예
```go
package main

import "fmt"

func main() {
    x := 10
    if x > 5 {
        fmt.Println("x는 5보다 큽니다.")
    } else {
        fmt.Println("x는 5보다 작거나 같습니다.")
    }
}
```

### 여러 조건 예
```go
package main

import "fmt"

func main() {
    x := 10
    if x < 0 {
        fmt.Println("x는 음수입니다.")
    } else if x == 0 {
        fmt.Println("x는 0입니다.")
    } else {
        fmt.Println("x는 양수입니다.")
    }
}
```

## 설명
- **중괄호의 필요성**: Go에서는 "if" 문에서 코드 블록을 정의하기 위해 반드시 중괄호를 사용해야 합니다. 이는 다른 언어와의 차이점으로, 코드의 가독성을 높이는 데 기여합니다.
- **조건의 평가**: 조건은 불리언 표현식으로 평가되어야 하며, 그렇지 않으면 컴파일 오류가 발생합니다.
- **변수의 범위**: "if" 문 내에서 선언된 변수는 블록 외부에서 접근할 수 없으므로 주의해야 합니다.

## 한 줄 요약
Go 언어의 "if" 문은 조건에 따라 코드 실행 흐름을 제어하는 강력한 도구입니다.
