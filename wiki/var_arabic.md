# استخدام الكلمة المفتاحية "var" في لغة البرمجة Go

## ملخص
تُستخدم الكلمة المفتاحية "var" في لغة البرمجة Go لتعريف المتغيرات. تساعد هذه الكلمة في تخصيص الذاكرة للمتغيرات وتحديد نوع البيانات التي يمكن تخزينها.

## الوثائق
تُعتبر "var" جزءًا أساسيًا من بناء جملة Go، حيث تُستخدم لتعريف المتغيرات في نطاقات مختلفة. يمكن استخدام "var" لتعريف متغيرات ذات نوع محدد أو نوع غير محدد، حيث يُمكن للغة Go استنتاج النوع من القيمة المُعطاة.

### الاستخدام
يتم استخدام "var" كالتالي:

```go
var variableName variableType
```

### مثال:
```go
var age int
var name string
```

يمكن أيضًا تعريف متغيرات متعددة في سطر واحد:
```go
var (
    height float64
    weight float64
)
```

## الأمثلة
### مثال 1: تعريف متغير بسيط
```go
package main

import "fmt"

func main() {
    var x int = 10
    fmt.Println(x)
}
```

### مثال 2: تعريف متغيرات متعددة
```go
package main

import "fmt"

func main() {
    var a, b int = 1, 2
    fmt.Println(a + b)
}
```

### مثال 3: تعريف متغير بدون تحديد النوع
```go
package main

import "fmt"

func main() {
    var name = "Ali"
    fmt.Println(name)
}
```

## الشرح
### نقاط يجب الانتباه لها:
1. **الأنواع الافتراضية**: عند تعريف متغير باستخدام "var" بدون إعطاء قيمة، يتم تعيين قيمة افتراضية للنمط المحدد (مثل 0 للأعداد و"" للسلاسل النصية).
2. **نطاق المتغيرات**: يجب الانتباه إلى نطاق المتغيرات، حيث يُمكن تعريف المتغيرات داخل الدوال أو خارجها، مما يؤثر على مدى وصولها.
3. **عدم التكرار**: لا يمكن تعريف متغير بنفس الاسم في نفس النطاق.

## ملخص من سطر واحد
تُستخدم الكلمة المفتاحية "var" في Go لتعريف المتغيرات وتخصيص أنواع البيانات في البرنامج.
