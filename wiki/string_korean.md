# Go 언어의 문자열(string) 처리: 기초부터 활용까지

## 개요
Go 언어에서 문자열(string)은 텍스트 데이터를 처리하는 기본 단위로, 다양한 기능과 메서드를 통해 문자열을 조작하고 관리할 수 있습니다. 이 문서에서는 Go 언어의 문자열에 대한 기본 개념, 사용법, 예제 및 주의할 점을 설명합니다.

## 문서화

### 목적
Go 언어의 문자열은 불변(immutable) 데이터 타입으로, 즉 한 번 생성된 문자열은 수정할 수 없습니다. 문자열은 UTF-8 인코딩을 사용하여 다양한 언어의 문자를 표현할 수 있으며, 다양한 내장 함수를 통해 문자열을 효과적으로 처리할 수 있습니다.

### 사용법
Go에서 문자열을 정의하려면 큰따옴표(`"`)를 사용합니다. 문자열은 `string` 타입으로 선언되며, 다음과 같은 기본적인 연산을 지원합니다:

- 문자열 길이: `len()` 함수를 사용하여 문자열의 길이를 구할 수 있습니다.
- 문자열 결합: `+` 연산자를 이용해 문자열을 결합할 수 있습니다.
- 문자열 슬라이스: 문자열의 특정 부분을 슬라이스를 통해 쉽게 추출할 수 있습니다.

### 세부사항
- **문자열 리터럴**: Go에서는 문자열 리터럴을 큰따옴표로 감싸서 정의합니다. 예를 들어, `s := "Hello, World!"`는 문자열 `s`에 "Hello, World!"를 할당합니다.
- **멀티라인 문자열**: 백틱(``` ` ```)을 사용하면 여러 줄에 걸친 문자열을 정의할 수 있습니다.
- **문자열과 바이트**: 문자열을 바이트 슬라이스로 변환하려면 `[]byte(s)`를 사용하며, 반대로는 `string(b)`로 바이트 슬라이스를 문자열로 변환할 수 있습니다.

## 예제

### 기본 사용 예제
```go
package main

import (
    "fmt"
)

func main() {
    // 문자열 선언
    greeting := "안녕하세요, Go 언어!"
    
    // 문자열 길이 출력
    fmt.Println("문자열 길이:", len(greeting))
    
    // 문자열 결합
    name := "사용자"
    welcomeMessage := greeting + " " + name
    fmt.Println(welcomeMessage)
    
    // 문자열 슬라이스
    fmt.Println("부분 문자열:", greeting[0:5]) // "안녕하"
}
```

### 멀티라인 문자열 예제
```go
package main

import "fmt"

func main() {
    multiline := `이것은
멀티라인
문자열입니다.`
    fmt.Println(multiline)
}
```

## 설명
문자열을 사용할 때 주의할 점은 문자열이 불변이라는 것입니다. 문자열을 직접 수정할 수 없기 때문에, 문자열을 변경하고 싶다면 새로운 문자열을 생성해야 합니다. 또한, 문자열의 인덱싱은 바이트 단위로 이루어지므로, UTF-8 인코딩을 고려하지 않으면 한글과 같은 다국어 문자 처리에서 오류가 발생할 수 있습니다. 이는 문자열 슬라이스를 할 때 특히 주의해야 합니다.

## 한 줄 요약
Go 언어의 문자열은 불변의 UTF-8 인코딩 텍스트 데이터로, 다양한 내장 함수를 이용해 효과적으로 조작할 수 있습니다.
