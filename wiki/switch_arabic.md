# استخدام عبارة "switch" في لغة Go: دليل شامل

## ملخص
تعتبر عبارة "switch" في لغة Go أداة قوية تسهل اتخاذ القرارات بناءً على قيمة معينة. تتيح للمطورين كتابة كود أكثر وضوحًا وتنظيمًا عند التعامل مع حالات متعددة.

## الوثائق
تُستخدم عبارة "switch" في Go لتحديد تنفيذ كتلة من التعليمات البرمجية بناءً على قيمة معينة. يمكن استخدامها كبديل لعبارة if-else المتعددة. 

### الغرض
الغرض من "switch" هو تبسيط الشروط المعقدة والسماح بالتحكم في تدفق البرنامج بطريقة مرتبة. 

### الاستخدام
تأتي عبارة "switch" بشكل أساسي في الصيغة التالية:

```go
switch expression {
case value1:
    // تنفيذ الكود إذا كانت القيمة تساوي value1
case value2:
    // تنفيذ الكود إذا كانت القيمة تساوي value2
default:
    // تنفيذ الكود إذا لم تطابق أي حالة
}
```

### التفاصيل
- **expression**: تعبير يتم تقييمه. يمكن أن يكون قيمة أو متغير.
- **case**: كل حالة تمثل قيمة ممكنة لـ expression. يمكن أن تحتوي على أكثر من قيمة مفصولة بفواصل.
- **default**: حالة اختيارية تُنفذ إذا لم تتطابق أي حالة مع expression.

يمكن أيضًا استخدام "switch" بدون تعبير، حيث سيتم استخدام القيمة الافتراضية لـ true:

```go
switch {
case condition1:
    // تنفيذ الكود إذا كانت condition1 صحيحة
case condition2:
    // تنفيذ الكود إذا كانت condition2 صحيحة
default:
    // تنفيذ الكود إذا لم تكن أي من الشروط صحيحة
}
```

## الأمثلة
### مثال 1: استخدام عبارة switch مع تعبير
```go
package main

import (
    "fmt"
)

func main() {
    day := 3
    switch day {
    case 1:
        fmt.Println("الإثنين")
    case 2:
        fmt.Println("الثلاثاء")
    case 3:
        fmt.Println("الأربعاء")
    default:
        fmt.Println("يوم غير معروف")
    }
}
```

### مثال 2: استخدام switch بدون تعبير
```go
package main

import (
    "fmt"
)

func main() {
    score := 85
    switch {
    case score >= 90:
        fmt.Println("ممتاز")
    case score >= 75:
        fmt.Println("جيد جداً")
    case score >= 60:
        fmt.Println("جيد")
    default:
        fmt.Println("رسوب")
    }
}
```

## الشرح
### الأخطاء الشائعة
1. **نسيان استخدام break**: في Go، لا حاجة لاستخدام جملة break، حيث تنتهي كل حالة تلقائيًا بعد تنفيذ الكود. لكن من المهم أن نفهم ذلك لتجنب الخلط مع لغات أخرى.
2. **تكرار القيم**: يجب أن تكون القيم في حالة "case" فريدة، وإلا سيظهر خطأ في وقت الترجمة.
3. **الترتيب**: يجب أن تكون الحالات مرتبة بشكل منطقي، حيث يتم تقييمها بالترتيب.

### ملاحظات إضافية
- يمكن استخدام عدة قيم في حالة واحدة. على سبيل المثال:
```go
switch day {
case 1, 2, 3:
    fmt.Println("أيام الأسبوع")
}
```
- يمكن استخدام نوع البيانات المخصصة في التعبيرات.

## ملخص جملة واحدة
عبارة "switch" في لغة Go هي أداة قوية لتنظيم اتخاذ القرارات بناءً على قيمة معينة، مما يسهل كتابة كود واضح ومرتب.
