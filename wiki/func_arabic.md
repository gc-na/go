# الدالة `func` في لغة Go: تعريف، استخدامات وأمثلة

## ملخص
تعتبر الكلمة المفتاحية `func` في لغة Go الوسيلة الأساسية لتعريف الدوال. تتيح لك هذه الكلمة كتابة وظائف قابلة لإعادة الاستخدام، مما يسهل تنظيم الكود ويعزز من قابلية الصيانة.

## الوثائق
تستخدم الكلمة المفتاحية `func` لتعريف دالة جديدة في لغة Go. يمكن أن تأخذ الدالة معلمات وتعيد قيمة. صيغة تعريف الدالة هي كما يلي:

```go
func اسم_الدالة(معلمات) نوع_القيمة_المرجعة {
    // تنفيذ الدالة
}
```

### الغرض
تستخدم الدوال لتقسيم الكود إلى وحدات صغيرة، مما يسهل فهمه وإعادة استخدامه. يمكن أن تحتوي الدالة على معلمات لتلقي المدخلات، ويمكن أن تعيد قيمًا مختلفة اعتمادًا على المنطق البرمجي.

### الاستخدام
لتعريف دالة، يجب عليك تحديد اسمها، والمعلمات (إن وجدت)، ونوع القيمة المرجعة (إن وجدت). إذا كانت الدالة لا تعيد أي قيمة، يمكنك استخدام الكلمة المفتاحية `return` بدون قيمة.

### التفاصيل
- يمكن أن تحتوي الدالة على معلمات متعددة، مفصولة بفواصل.
- يمكن تعريف الدوال داخل حزم (Packages) أو كمكونات مستقلة.
- يمكن أن تكون الدوال متداخلة، حيث يمكن أن تحتوي دالة على دوال أخرى داخلها.

## أمثلة
### مثال 1: دالة بسيطة بدون معلمات
```go
package main

import "fmt"

func sayHello() {
    fmt.Println("مرحبًا بالعالم!")
}

func main() {
    sayHello()
}
```

### مثال 2: دالة مع معلمات
```go
package main

import "fmt"

func add(a int, b int) int {
    return a + b
}

func main() {
    result := add(3, 5)
    fmt.Println("النتيجة:", result)
}
```

### مثال 3: دالة بدون قيمة مرجعة
```go
package main

import "fmt"

func printMessage(message string) {
    fmt.Println(message)
}

func main() {
    printMessage("مرحبًا بك في Go!")
}
```

## الشرح
عند استخدام الكلمة المفتاحية `func`، يجب الانتباه إلى بعض النقاط:
- **أسماء الدوال**: يجب أن تكون فريدة ضمن الحزمة. يفضل استخدام أسماء تعبر عن وظيفتها.
- **عدد المعلمات**: تأكد من تحديد أنواع المعلمات بشكل صحيح. إذا تم تمرير نوع غير صحيح، ستظهر أخطاء أثناء الترجمة.
- **القيمة المرجعة**: إذا كانت دالتك تحتاج إلى إرجاع قيمة، تأكد من استخدام الكلمة المفتاحية `return`.

## ملخص جملة واحدة
الكلمة المفتاحية `func` في Go تُستخدم لتعريف الدوال، مما يسهل كتابة كود منظم وقابل لإعادة الاستخدام.
