# استخدام عبارة "for" في لغة Go: الشرح والأمثلة

## ملخص
عبارة "for" في لغة Go تُستخدم لإنشاء حلقات تكرارية، مما يُتيح للمطورين تنفيذ كود معين عدة مرات بناءً على شروط محددة. تُعتبر هذه العبارة من العناصر الأساسية في لغة Go وتعتبر أداة قوية في برمجة التطبيقات.

## الوثائق
تُستخدم عبارة "for" في لغة Go لتنفيذ حلقات تكرارية. يمكن استخدامها بعدة طرق، منها:

1. **حلقة بسيطة**: تتكرر حتى يتحقق شرط معين.
2. **حلقة مع عداد**: تستخدم عدادًا لتمرير القيم.
3. **حلقة غير محدودة**: تستمر إلى ما لا نهاية حتى يتم الخروج منها باستخدام عبارة `break`.

### الاستخدام
تتبع عبارة "for" الصيغة التالية:

```go
for condition {
    // الكود الذي سيتم تنفيذه
}
```

### تفاصيل إضافية
- **الحلقات مع العدادات**: يمكن استخدام المتغيرات كعدادات:
  
  ```go
  for i := 0; i < 10; i++ {
      // الكود الذي سيتم تنفيذه
  }
  ```

- **الحلقات غير المحدودة**: يمكن استخدام `for` بدون شروط للخروج منها:
  
  ```go
  for {
      // الكود الذي سيتم تنفيذه
      if condition {
          break
      }
  }
  ```

- **استخدام `range`**: يمكن استخدام `for` مع `range` للتكرار على المصفوفات أو الشرائح:
  
  ```go
  for index, value := range array {
      // الكود الذي سيتم تنفيذه
  }
  ```

## الأمثلة

### مثال 1: حلقة بسيطة
```go
package main

import "fmt"

func main() {
    i := 0
    for i < 5 {
        fmt.Println(i)
        i++
    }
}
```

### مثال 2: حلقة مع عداد
```go
package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}
```

### مثال 3: حلقة غير محدودة
```go
package main

import "fmt"

func main() {
    count := 0
    for {
        fmt.Println(count)
        count++
        if count == 5 {
            break
        }
    }
}
```

### مثال 4: استخدام `range`
```go
package main

import "fmt"

func main() {
    slice := []string{"apple", "banana", "cherry"}
    for index, value := range slice {
        fmt.Printf("Index: %d, Value: %s\n", index, value)
    }
}
```

## الشرح
من الشائع أن يقع المبرمجون في بعض الأخطاء عند استخدام عبارة "for"، مثل عدم تحديث العدادات بشكل صحيح، مما يؤدي إلى حلقات غير منتهية. تأكد دائمًا من أن الشروط التي تستخدمها في الحلقة ستؤدي في النهاية إلى الخروج منها. أيضًا، استخدام الحلقات غير المحدودة يجب أن يكون مدروسًا جيدًا لتجنب استهلاك الموارد بشكل غير ضروري.

## ملخص في جملة واحدة
عبارة "for" في لغة Go تُستخدم لإنشاء حلقات تكرارية قوية ومرنة، مما يسمح بتنفيذ كود معين عدة مرات بناءً على شروط محددة.
