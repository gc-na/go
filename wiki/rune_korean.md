# Go 언어의 rune: 문자 및 유니코드 처리

## 개요
Go 언어에서 `rune`은 유니코드 문자 하나를 나타내는 데이터 타입입니다. `rune`은 Go의 기본적인 문자 처리 방법으로, 문자열 및 문자의 다양한 조작을 가능하게 합니다.

## 문서화

### 목적
`rune`은 Go에서 유니코드 문자를 처리하기 위해 설계된 타입으로, `int32`와 동일한 크기를 가지며, 음수와 양수를 모두 표현할 수 있습니다. 이는 다양한 언어와 기호를 포함한 문자열을 효과적으로 다룰 수 있도록 합니다.

### 사용법
`rune`은 기본적으로 문자열에서 특정 문자를 추출하거나, 문자의 코드를 확인하는 데 사용됩니다. 다음은 `rune` 타입을 정의하고 사용하는 방법의 예입니다.

```go
var r rune = 'a' // 소문자 'a'를 rune으로 저장
fmt.Println(r)   // 출력: 97 (ASCII 코드)
```

문자열을 `rune` 슬라이스로 변환하면 문자열의 각 문자를 개별적으로 처리할 수 있습니다.

```go
s := "안녕하세요"
runes := []rune(s)
fmt.Println(runes) // 출력: [50504 45397 54616 50836 50857] (각 문자의 유니코드 값)
```

### 세부 사항
`rune`을 사용할 때 주의할 점은, `rune`이 4바이트 크기를 가지므로, ASCII와 같은 1바이트 문자와는 다르게 처리된다는 점입니다. 이는 여러 바이트로 이루어진 문자를 사용할 때, 직접적으로 인덱스를 통해 접근할 경우 예상치 못한 결과를 초래할 수 있습니다.

## 예시

### 기본 사용 예
```go
package main

import (
    "fmt"
)

func main() {
    var r rune = 'A'
    fmt.Printf("rune: %c, code: %d\n", r, r) // 출력: rune: A, code: 65
}
```

### 문자열을 rune 슬라이스로 변환하기
```go
package main

import (
    "fmt"
)

func main() {
    str := "Hello, 世界"
    runes := []rune(str)
    for i, r := range runes {
        fmt.Printf("index: %d, rune: %c, code: %d\n", i, r, r)
    }
}
```

## 설명
`rune`을 사용할 때 흔히 겪는 문제는 문자열의 길이와 `rune`의 개수가 다르다는 점입니다. 문자열의 길이는 바이트 단위로 계산되기 때문에, 다국어 문자열을 사용할 때 인덱스에 따라 잘못된 문자를 참조할 수 있습니다.

또한, `rune` 타입은 문자 연산을 수행할 때 간편하게 사용할 수 있지만, 이 타입을 사용할 때 문자와의 연관성을 이해하는 것이 중요합니다. 특히, 문자열을 조작할 때는 항상 `rune` 슬라이스를 사용해야 합니다.

## 한 줄 요약
Go 언어에서 `rune`은 유니코드 문자를 표현하고 조작하기 위한 타입으로, 다양한 문자와 기호를 효과적으로 처리할 수 있도록 돕습니다.
