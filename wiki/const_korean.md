# Go 언어의 "const": 상수 선언 및 사용법

## 개요
Go 언어에서 `const`는 상수를 선언하는 데 사용됩니다. 상수는 프로그램 실행 중에 변하지 않는 고정된 값을 나타내며, 코드의 가독성을 높이고, 유지보수를 용이하게 합니다.

## 문서화
`const` 키워드는 Go에서 상수를 정의하는 데 사용됩니다. 상수는 숫자, 문자열, 불리언 등의 데이터 타입을 가질 수 있으며, 한 번 정의된 상수는 변경할 수 없습니다. 상수를 사용하면 코드의 의미를 명확히 하고, 다양한 값들을 쉽게 관리할 수 있습니다.

### 목적
- 변하지 않는 값들을 정의하여 코드의 안정성을 높입니다.
- 프로그램의 의도를 명확히 하고, 가독성을 향상시킵니다.

### 사용법
상수를 선언할 때는 `const` 키워드를 사용하고, 다음과 같은 형식으로 작성합니다:

```go
const 상수이름 데이터타입 = 값
```

여러 개의 상수를 선언할 때는 괄호를 사용할 수 있습니다:

```go
const (
    상수이름1 데이터타입 = 값1
    상수이름2 데이터타입 = 값2
)
```

## 예제
다음은 Go에서 상수를 사용하는 간단한 예제입니다.

```go
package main

import "fmt"

const PI = 3.14
const Greeting = "안녕하세요"

func main() {
    fmt.Println("원의 면적:", PI * 5 * 5)
    fmt.Println(Greeting)
}
```

위 예제에서 `PI`와 `Greeting`은 각각 원주율과 인사말을 나타내는 상수입니다.

## 설명
상수를 사용할 때 주의해야 할 몇 가지 사항이 있습니다:

- **타입**: 상수는 명시적으로 타입을 지정하지 않으면 기본값을 가집니다. 따라서, 타입을 명확히 지정하는 것이 좋습니다.
- **범위**: 상수의 범위는 선언된 블록 내에서만 유효합니다. 따라서 전역 상수와 지역 상수를 구분해야 합니다.
- **복합 상수**: 상수는 다른 상수와 조합하여 사용할 수 있으며, 이 경우 계산된 값도 상수가 됩니다.

상수를 잘못 사용하면 코드의 의도를 오해할 수 있으므로, 의미 있는 이름을 부여하고 주석을 추가하는 것이 좋습니다.

## 한 줄 요약
Go 언어의 `const`는 변하지 않는 상수를 선언하여 코드의 가독성과 안정성을 높이는 기능입니다.
