# Go: Go 언어의 기본 명령어 및 사용법

## 개요
"Go"는 Google에서 개발한 프로그래밍 언어로, 단순성과 성능을 결합하여 효율적인 소프트웨어 개발을 지원합니다. 이 문서에서는 Go 언어의 기본 개념과 사용법을 설명합니다.

## 문서화

### 목적
Go 언어는 시스템 프로그래밍 및 서버 사이드 애플리케이션 개발을 위해 설계되었습니다. 고루틴(goroutines)과 채널(channels)과 같은 동시성 개념을 제공하여, 높은 성능과 효율성을 자랑합니다.

### 사용법
Go는 명령어를 사용하여 프로그램을 컴파일하고 실행합니다. 기본적인 사용 방법은 다음과 같습니다:

1. **Go 설치**: Go 언어를 사용하기 위해서는 먼저 Go를 설치해야 합니다. 공식 웹사이트(https://golang.org/dl/)에서 최신 버전을 다운로드할 수 있습니다.
2. **환경 설정**: 설치 후, GOPATH 및 GOROOT 환경 변수를 설정해야 합니다.
3. **프로그램 작성**: `.go` 확장자를 가진 파일을 생성하고 Go 코드를 작성합니다.
4. **컴파일 및 실행**: `go run <파일명.go>` 명령어를 사용하여 프로그램을 실행합니다.

### 세부 사항
Go 언어는 다음과 같은 주요 기능을 제공합니다:

- **정적 타입 언어**: 변수의 타입이 컴파일 시에 결정됩니다.
- **고루틴**: 경량 스레드를 사용하여 동시성을 쉽게 구현할 수 있습니다.
- **패키지 관리**: `go get` 명령어를 통해 외부 패키지를 쉽게 관리할 수 있습니다.
- **가비지 컬렉션**: 메모리 관리를 자동으로 처리하여 개발자가 메모리 누수 문제를 걱정하지 않도록 합니다.

## 예시

### 기본 Hello World 프로그램
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
위 코드를 `hello.go` 파일에 저장한 후, 터미널에서 다음 명령어를 입력하여 실행할 수 있습니다:
```bash
go run hello.go
```

### 고루틴 사용 예
```go
package main

import (
    "fmt"
    "time"
)

func sayHello() {
    fmt.Println("Hello from Goroutine")
}

func main() {
    go sayHello()
    time.Sleep(1 * time.Second) // 고루틴이 실행될 시간을 주기 위해 잠시 대기
}
```
위 예제에서는 `sayHello` 함수를 고루틴으로 실행합니다.

## 설명
Go 언어를 사용할 때 주의할 점은 다음과 같습니다:

- **동시성**: 고루틴을 사용할 때, 프로그램의 흐름을 충분히 이해하고 있어야 합니다. 잘못된 동기화는 데이터 경쟁을 초래할 수 있습니다.
- **패키지 관리**: 패키지를 추가할 때 `go get` 명령어를 사용해야 하며, 올바른 import 경로를 지정해야 합니다.
- **가비지 컬렉션**: 메모리 관리를 자동으로 처리하지만, 참조를 잘못 관리하면 메모리 누수가 발생할 수 있습니다.

## 한 줄 요약
Go는 효율적인 소프트웨어 개발을 위한 정적 타입의 프로그래밍 언어로, 고루틴을 통해 동시성을 지원합니다.
