# Go语言中的“return”语句详解

## 摘要
“return”是Go语言中用于结束函数执行并返回指定值的重要语句。它在控制程序流和处理函数结果时起着关键作用。

## 文档
### 目的
“return”语句用于从函数中返回值或结束函数执行。它可以返回一个或多个值，也可以不返回任何值。通过“return”，程序可以将函数计算的结果传递给调用函数。

### 用法
“return”语句的基本语法如下：
```go
return [返回值]
```
- **返回值**: 可以是一个表达式或多个表达式，具体取决于函数的定义。

### 详细说明
- **在函数中使用**: “return”语句只能在函数体内部使用。它标志着函数的结束，并且可以选择性地返回值。
- **多个返回值**: Go语言支持多个返回值，函数定义时需要指定返回值的类型。例如：
  ```go
  func add(a int, b int) (int, int) {
      return a + b, a - b
  }
  ```
- **无返回值函数**: 如果函数不需要返回值，可以省略“return”或仅使用“return”而不带任何值：
  ```go
  func noReturn() {
      return
  }
  ```

## 示例
以下是一些“return”语句的基本用法示例：

### 示例1：简单返回值
```go
func square(x int) int {
    return x * x
}
```

### 示例2：多个返回值
```go
func divide(x, y int) (int, int) {
    return x / y, x % y
}
```

### 示例3：无返回值函数
```go
func printMessage(message string) {
    fmt.Println(message)
    return
}
```

## 说明
- **未使用的返回值警告**: 如果函数返回值未被使用，编译器会发出警告。这通常是由于开发者忘记使用返回结果。
- **多重返回值处理**: 在调用返回多个值的函数时，必须正确处理所有返回值。如果只想使用其中一个，可以使用下划线`_`来忽略不需要的值：
  ```go
  _, remainder := divide(10, 3)
  ```
- **控制流**: 使用“return”可以有效地控制程序流，尤其是在处理错误时。通过在错误发生时立即返回，可以避免执行后续代码。

## 一句话总结
“return”语句在Go语言中用于结束函数执行并返回值，是控制程序流和处理函数结果的关键工具。
