# Go 語言中的「if」條件語句：基礎與實踐

## 摘要
「if」語句是 Go 語言中用來實現條件判斷的基本控制結構。它允許程序根據不同的條件執行不同的代碼塊，從而增強了程序的靈活性。

## 文檔
在 Go 語言中，「if」語句的主要目的是根據表達式的真偽來控制程式的執行流程。語法如下：

```go
if 條件 {
    // 當條件為真時執行的代碼
}
```

### 使用方法
1. **基本結構**：如果條件為真，則執行大括號內的代碼。
2. **else 語句**：可以跟隨一個 `else` 語句來處理條件為假時的情況。
3. **else if**：可以使用 `else if` 來處理多個條件。

### 詳細說明
- **條件表達式**：必須返回一個布爾值（`true` 或 `false`）。
- **可選初始化語句**：可以在 `if` 語句中加入初始化語句，用於定義變量。

例如：

```go
if x := 10; x > 5 {
    fmt.Println("x 大於 5")
}
```

在這個例子中，變量 `x` 只在 `if` 語句的範圍內可用。

## 示例
以下是 Go 語言中「if」語句的基本用法示例：

### 基本用法
```go
package main

import "fmt"

func main() {
    num := 10
    if num > 5 {
        fmt.Println("數字大於 5")
    }
}
```

### 使用 else
```go
package main

import "fmt"

func main() {
    num := 3
    if num > 5 {
        fmt.Println("數字大於 5")
    } else {
        fmt.Println("數字不大於 5")
    }
}
```

### 使用 else if
```go
package main

import "fmt"

func main() {
    num := 5
    if num > 5 {
        fmt.Println("數字大於 5")
    } else if num == 5 {
        fmt.Println("數字等於 5")
    } else {
        fmt.Println("數字小於 5")
    }
}
```

## 解釋
在使用「if」語句時，有幾個常見的陷阱需要注意：
- **條件表達式的類型**：確保條件表達式返回的是布爾值，否則會導致編譯錯誤。
- **作用域問題**：如果在 `if` 語句中聲明變量，該變量的作用域僅限於該語句內。
- **嵌套 `if`**：可以在 `if` 語句內部再使用 `if`，但要注意代碼的可讀性。

## 一句總結
Go 語言中的「if」條件語句是控制程序流程的基本工具，可根據條件執行不同的代碼塊。
