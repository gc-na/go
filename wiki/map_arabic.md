# الماب في لغة جوجل Go: كل ما تحتاج معرفته

## الملخص
الماب في لغة Go هو نوع بيانات يتيح تخزين أزواج من القيم، مما يسهل الوصول إلى القيم من خلال مفاتيحها. يعتبر الماب من الهياكل الأساسية في لغة Go ويستخدم بشكل واسع في البرمجة.

## الوثائق
الماب (map) في Go هو نوع بيانات غير مرتب يستخدم لتخزين أزواج من المفاتيح والقيم. يمكن الوصول إلى القيمة المرتبطة بكل مفتاح بسرعة، مما يجعل الماب خيارًا مثاليًا لتخزين البيانات التي تتطلب عمليات بحث سريعة. 

### الهدف
الغرض من استخدام الماب هو توفير طريقة فعالة لتخزين واسترجاع البيانات. يمكن استخدام الماب لتخزين أي نوع من البيانات، سواء كانت أعداد صحيحة، نصوص، أو حتى هياكل بيانات مخصصة.

### الاستخدام
لإنشاء ماب، يمكنك استخدام الكلمة المفتاحية `make` أو استخدام القوسين المعقفين `{}`. إليك كيفية تعريف ماب:

```go
myMap := make(map[string]int)
```

أو:

```go
myMap := map[string]int{"key1": 1, "key2": 2}
```

يمكنك إضافة أو تعديل قيمة باستخدام المفتاح:

```go
myMap["key3"] = 3
```

للوصول إلى قيمة معينة باستخدام المفتاح:

```go
value := myMap["key1"]
```

لإزالة مفتاح وقيمته:

```go
delete(myMap, "key2")
```

### التفاصيل
- **مفاتيح فريدة**: يجب أن تكون المفاتيح فريدة داخل الماب. إذا قمت بإضافة مفتاح موجود بالفعل، سيتم استبدال قيمته.
- **أنواع المفاتيح والقيم**: يمكن أن تكون المفاتيح من أنواع بسيطة مثل السلاسل أو الأعداد، لكن لا يمكن أن تكون من أنواع مثل الشرائح أو الماب نفسها.
- **حجم الماب**: يمكن أن يتغير حجم الماب ديناميكيًا مع مرور الوقت، مما يوفر مرونة في استخدامه.

## الأمثلة
### مثال 1: إنشاء ماب وإضافة القيم
```go
package main

import "fmt"

func main() {
    myMap := make(map[string]int)
    myMap["apple"] = 5
    myMap["banana"] = 10
    fmt.Println(myMap) // Output: map[apple:5 banana:10]
}
```

### مثال 2: الوصول إلى القيم
```go
value := myMap["apple"]
fmt.Println(value) // Output: 5
```

### مثال 3: حذف قيمة
```go
delete(myMap, "banana")
fmt.Println(myMap) // Output: map[apple:5]
```

## الشرح
### الأخطاء الشائعة
- **استخدام مفتاح غير موجود**: إذا قمت بمحاولة الوصول إلى قيمة باستخدام مفتاح غير موجود، ستحصل على القيمة الصفرية لنوع القيمة (مثل 0 للأعداد).
- **عدم استخدام نوع بيانات مناسب للمفتاح**: تأكد من أن المفاتيح المستخدمة في الماب هي من الأنواع المدعومة لتجنب الأخطاء.
- **تعديل الماب أثناء التكرار**: تجنب تعديل الماب أثناء التكرار عليه، لأن ذلك قد يؤدي إلى سلوك غير متوقع.
