# الهيكل (Struct) في لغة Go: تعريف واستخدامات

## ملخص
الهيكل (Struct) في لغة Go هو نوع بيانات مُركب يُستخدم لتجميع عدة قيم ذات أنواع مختلفة تحت اسم واحد، مما يسهل تنظيم البيانات وتعزيز قابلية القراءة والصيانة في البرمجيات.

## الوثائق
### الغرض
الهيكل (Struct) هو نوع بيانات مخصص يُستخدم في لغة Go لتعريف كائنات تحتوي على مجموعة من الخصائص. يُمكن استخدام الهياكل لتعريف أنواع جديدة وتطبيقها في مجالات مختلفة مثل قواعد البيانات، واجهات المستخدم، وغيرها.

### الاستخدام
يتم تعريف الهيكل باستخدام الكلمة المفتاحية `type` متبوعةً باسم الهيكل والكلمة المفتاحية `struct`. يمكن أن يحتوي الهيكل على حقول (fields) من أنواع مختلفة. إليك كيفية تعريف هيكل:

```go
type Person struct {
    Name string
    Age  int
}
```

### التفاصيل
- **المجالات (Fields)**: يمكن أن تحتوي على أي نوع من البيانات، بما في ذلك الأنواع البدائية مثل الأعداد الصحيحة والسلاسل النصية، بالإضافة إلى هياكل أخرى.
- **الترتيب**: ترتيب الحقول مهم في الهيكل، حيث يُستخدم في بعض المواقف لتحديد كيفية ترتيب البيانات في الذاكرة.
- **التحكم في الوصول**: إذا كان اسم الحقل يبدأ بحرف كبير، فإنه يصبح متاحًا خارج الحزمة (public)، وإذا بدأ بحرف صغير، فإنه يكون خاصًا بالحزمة (private).

## الأمثلة
### تعريف واستخدام هيكل بسيط
```go
package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

func main() {
    person1 := Person{Name: "علي", Age: 30}
    fmt.Println(person1.Name, person1.Age)
}
```

### هيكل يحتوي على هيكل آخر
```go
package main

import "fmt"

type Address struct {
    City    string
    Country string
}

type Person struct {
    Name    string
    Age     int
    Address Address
}

func main() {
    person1 := Person{
        Name: "علي",
        Age:  30,
        Address: Address{
            City:    "القاهرة",
            Country: "مصر",
        },
    }
    fmt.Println(person1)
}
```

## الشرح
### المشاكل الشائعة
- **عدم تهيئة الحقول**: إذا لم يتم تهيئة الحقول، ستأخذ القيم الافتراضية لنوع البيانات. على سبيل المثال، السلاسل النصية ستكون فارغة (""), والأعداد الصحيحة ستكون 0.
- **تداخل الهياكل**: عند استخدام هياكل تحتوي على هياكل أخرى، يجب التأكد من تهيئة جميع الحقول بشكل صحيح لضمان عدم وجود بيانات غير صحيحة.

### ملاحظات إضافية
- يمكن استخدام الهياكل كنوع مرجعي، مما يعني أنه يمكن تمريرها كمرجع إلى الدوال.
- يمكن استخدام الهياكل في تحقيق واجهات (interfaces) أو كنوع بيانات في الحزم.

## ملخص من سطر واحد
الهيكل (Struct) في لغة Go هو وسيلة قوية لجمع البيانات ذات الأنواع المختلفة في نوع واحد منظم وسهل الاستخدام.
