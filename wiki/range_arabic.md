# استخدام "range" في لغة Go: الدليل الشامل

## ملخص
تُستخدم الكلمة الرئيسية "range" في لغة Go للتكرار عبر القوائم والمصفوفات والخرائط، مما يسهل عملية الوصول إلى العناصر في تلك الهياكل.

## الوثائق
تعتبر "range" أداة قوية في Go تُستخدم لتسهيل عملية التكرار على أنواع مختلفة من البيانات. يمكن استخدامها مع:

- **المصفوفات (Arrays)**: للتكرار عبر العناصر.
- **القوائم (Slices)**: للوصول إلى العناصر.
- **الخرائط (Maps)**: للوصول إلى الأزواج (key-value).
- **القنوات (Channels)**: لقراءة القيم المرسلة عبر القناة.

### الاستخدام
يمكن استخدام "range" بطريقتين رئيسيتين:

1. **التكرار على المصفوفات والقوائم**:
   ```go
   for index, value := range collection {
       // استخدم index و value
   }
   ```
   حيث `collection` يمكن أن تكون مصفوفة أو قائمة.

2. **التكرار على الخرائط**:
   ```go
   for key, value := range myMap {
       // استخدم key و value
   }
   ```
   حيث `myMap` هي خريطة تحتوي على أزواج (key-value).

## أمثلة

### مثال على استخدام "range" مع القوائم
```go
package main

import "fmt"

func main() {
    fruits := []string{"تفاح", "موز", "برتقال"}
    for index, fruit := range fruits {
        fmt.Printf("الفاكهة رقم %d هي %s\n", index, fruit)
    }
}
```

### مثال على استخدام "range" مع الخرائط
```go
package main

import "fmt"

func main() {
    ages := map[string]int{"علي": 30, "فاطمة": 25}
    for name, age := range ages {
        fmt.Printf("%s عمره %d\n", name, age)
    }
}
```

### مثال على استخدام "range" مع القنوات
```go
package main

import "fmt"

func main() {
    ch := make(chan string)
    go func() {
        ch <- "مرحبا"
        ch <- "وداعا"
        close(ch)
    }()

    for msg := range ch {
        fmt.Println(msg)
    }
}
```

## الشرح
هناك بعض النقاط الهامة التي يجب مراعاتها عند استخدام "range":

- عند استخدامها مع الخرائط، يختلف ترتيب العناصر في كل مرة يتم فيها التكرار.
- عند استخدام "range" مع القوائم، يتم إنشاء نسخة من القيمة. إذا كنت بحاجة إلى تعديل القيم، يجب استخدام المؤشر.
- إذا لم تكن بحاجة إلى قيمة أو مفتاح، يمكنك استخدام الرمز `_` لتجاهلها.

## ملخص جملة واحدة
"range" هي أداة قوية في Go تسهل عملية التكرار عبر الهياكل المختلفة مثل القوائم والخرائط.
