# استخدام الأمر "break" في لغة Go: الدليل الشامل

## ملخص
الأمر "break" في لغة Go يُستخدم للخروج من حلقات التكرار مثل `for` و`switch`. يوفر هذا الأمر وسيلة للتحكم في تدفق البرنامج، مما يساعد المبرمجين على إنهاء التكرار عند تحقيق شرط معين.

## الوثائق
### الغرض
يُستخدم الأمر "break" لإنهاء الحلقة الحالية أو الخروج من الجملة `switch`. يمكن أن يكون الأمر مفيدًا في التحكم في تدفق البرنامج عند الحاجة إلى إنهاء عملية التكرار بناءً على شرط معين.

### الاستخدام
يتم استخدام "break" في سياقات مختلفة في Go، مثل حلقات `for` و`switch`. عند استخدام الأمر "break"، يتم إنهاء تنفيذ الحلقة أو الجملة الانتقالية بشكل فوري، ويتابع البرنامج تنفيذ التعليمات التالية بعد الحلقة أو الجملة.

### التفاصيل
- **في حلقات `for`**: يُستخدم "break" للخروج من الحلقة، سواء كانت حلقة تقليدية أو حلقة تعتمد على شرط. 
- **في جمل `switch`**: يُستخدم "break" لإنهاء حالة معينة داخل `switch`، ولكن في Go، لا تحتاج عادةً إلى استخدام "break" في `switch` لأنه يتم الخروج تلقائيًا بعد تنفيذ الحالة.
  
## الأمثلة
### مثال 1: استخدام break في حلقة for
```go
package main

import "fmt"

func main() {
    for i := 0; i < 10; i++ {
        if i == 5 {
            break
        }
        fmt.Println(i)
    }
}
```
**النتيجة:** سيظهر الأرقام من 0 إلى 4، ويتوقف عند الرقم 5.

### مثال 2: استخدام break في جملة switch
```go
package main

import "fmt"

func main() {
    day := "السبت"
    switch day {
    case "الأحد":
        fmt.Println("بداية الأسبوع")
    case "السبت":
        break // لا حاجة هنا بشكل صريح، يتم الخروج تلقائيًا
    default:
        fmt.Println("يوم عادي")
    }
}
```
**النتيجة:** لن يتم طباعة أي شيء لأن الحالة "السبت" لا تتطلب تنفيذ أي تعليمات أخرى.

## الشرح
### الأخطاء الشائعة
- **عدم استخدام break بشكل صحيح**: يجب أن تكون على دراية بمكان استخدام "break" لتجنب إنهاء الحلقة بشكل غير متوقع.
- **التباس مع حلقات متعددة**: عند استخدامه داخل حلقات متداخلة، قد يكون من الصعب تحديد أي حلقة ستخرج منها، لذا من الأفضل استخدام تسميات (labels) لتوضيح ذلك.

### ملاحظات إضافية
- في Go، لا يُستخدم "break" في جمل `switch` بنفس الطريقة التي يتم استخدامها في لغات أخرى مثل C أو Java، حيث يُخرج "break" من الحالة الحالية فقط، مما يجعل الكود أكثر وضوحًا.

## ملخص في جملة واحدة
الأمر "break" في Go يُستخدم للتحكم في تدفق البرنامج من خلال إنهاء الحلقات والجمل الانتقالية عند الحاجة.
