# Go 언어의 new: 메모리 할당과 객체 생성

## 개요
Go 언어에서 `new`는 메모리 할당을 통해 포인터를 반환하는 내장 함수입니다. 이 함수는 주로 구조체와 같은 사용자 정의 타입의 인스턴스를 생성할 때 사용됩니다.

## 문서화
`new` 함수는 특정 타입의 메모리를 할당하고, 해당 타입의 제로 값으로 초기화된 포인터를 반환합니다. 주로 다음과 같은 목적을 가지고 사용됩니다:

- **메모리 할당**: 새로운 메모리 블록을 할당하여 객체를 생성합니다.
- **제로 값 초기화**: 할당된 메모리는 해당 타입의 제로 값으로 자동 초기화됩니다.
  
### 사용법
`new` 함수의 기본 문법은 다음과 같습니다:

```go
p := new(T)
```

여기서 `T`는 메모리를 할당할 타입입니다. `p`는 타입 `*T`의 포인터가 됩니다.

### 세부사항
- `new`는 기본적으로 타입의 포인터를 반환합니다. 따라서, `new`로 생성된 변수는 사용하기 전에 반드시 역참조해야 합니다.
- `new`는 구조체, 배열, 슬라이스, 맵 등 다양한 타입에 사용할 수 있습니다.
- `new`를 사용할 경우, 반드시 타입의 제로 값이 유용한 경우에 사용해야 합니다.

## 예제
아래의 예제는 `new`를 사용하여 구조체 인스턴스를 생성하는 방법을 보여줍니다.

```go
package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

func main() {
    p := new(Person)
    p.Name = "Alice"
    p.Age = 30

    fmt.Println(*p) // 출력: {Alice 30}
}
```

위 코드에서 `new(Person)`은 `Person` 타입의 포인터를 생성하고, 그 포인터를 통해 구조체의 필드에 접근합니다.

## 설명
`new`를 사용할 때 주의해야 할 몇 가지 사항은 다음과 같습니다:

1. **포인터 반환**: `new`는 항상 포인터를 반환하므로, 값을 직접 사용하려면 역참조(`*`)가 필요합니다.
2. **메모리 관리**: Go는 가비지 컬렉션을 지원하지만, 불필요한 메모리 할당을 피하는 것이 좋습니다. 필요하지 않은 객체는 적시에 가비지 컬렉터가 회수할 수 있도록 관리해야 합니다.
3. **제로 값**: 생성된 객체의 필드는 제로 값으로 초기화됩니다. 이 점을 활용하여 초기값을 설정할 수 있습니다.

## 한 줄 요약
Go 언어의 `new` 함수는 메모리를 할당하고 제로 값으로 초기화된 타입의 포인터를 반환하는 기능입니다.
