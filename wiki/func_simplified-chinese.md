# Go语言中的函数（func）用法详解

## 概述
在Go语言中，`func`是用于定义函数的关键字。函数是Go编程的基础组成部分，允许开发者封装代码逻辑，以便重复使用和提高可读性。

## 文档
### 目的
`func`关键字用于声明一个函数，函数可以接收参数并返回值。它是组织代码、实现模块化和提高代码复用的重要工具。

### 使用方法
函数的基本语法如下：
```go
func 函数名(参数列表) 返回值类型 {
    // 函数体
}
```
- **函数名**：标识函数的名称，遵循标识符命名规则。
- **参数列表**：用括号括起来的零个或多个参数，每个参数由类型和名称组成，多个参数用逗号分隔。
- **返回值类型**：函数返回值的类型，可以是单个类型，也可以是多个类型，多个类型用括号括起来。

### 示例
以下是一个简单的函数示例，计算两个整数的和：

```go
package main

import "fmt"

// 定义一个函数，接收两个整数参数，返回它们的和
func add(a int, b int) int {
    return a + b
}

func main() {
    result := add(3, 5)
    fmt.Println("3 + 5 =", result) // 输出：3 + 5 = 8
}
```

在上述示例中，`add`函数接收两个整数并返回它们的和。

## 解释
### 常见陷阱
- **命名冲突**：函数名必须唯一，避免与其他函数或变量冲突。
- **参数类型**：参数类型必须明确，虽然可以在参数列表中省略参数名称，但类型必须指定。
- **返回值处理**：调用函数时，需注意处理返回值，特别是当函数返回多个值时。

### 其他注意事项
- 函数可以是高阶函数，即将其他函数作为参数或返回函数。
- Go语言不支持方法重载，但可以通过不同的参数类型或数量实现相似的功能。
- 函数可以嵌套定义，即在一个函数内部定义另一个函数。

## 一句话总结
在Go语言中，`func`关键字用于定义函数，是实现模块化和代码复用的核心机制。
