# Go 언어에서 int64 타입 이해하기

## 개요
Go 언어에서 `int64`는 64비트 정수형 데이터 타입으로, 높은 정밀도를 요구하는 수치 계산에 적합합니다. 이 타입은 메모리와 성능 측면에서 효율적이며, 다양한 수치 연산에 용이하게 사용됩니다.

## 문서화
`int64`는 Go 언어의 기본 데이터 타입 중 하나로, 64비트 정수를 표현합니다. 이 타입은 -2^63에서 2^63-1 범위의 값을 가질 수 있으며, 주로 큰 수치를 다룰 때 사용됩니다. `int64`는 `int` 타입의 크기와는 무관하게 항상 64비트로 정의되어 있어, 시스템에 관계없이 동일한 범위를 제공합니다.

### 사용법
`int64` 타입은 변수를 선언할 때 다음과 같이 사용할 수 있습니다:

```go
var number int64 = 9223372036854775807
```

또는 타입 변환을 통해 다른 정수형에서 `int64`로 변환할 수 있습니다:

```go
var smallNumber int = 100
var bigNumber int64 = int64(smallNumber)
```

## 예제
다음은 `int64` 타입을 사용하는 몇 가지 기본 예제입니다.

### 예제 1: 기본 사용
```go
package main

import "fmt"

func main() {
    var a int64 = 10000000000
    var b int64 = 20000000000
    var sum int64 = a + b
    fmt.Println("Sum:", sum) // 출력: Sum: 30000000000
}
```

### 예제 2: 타입 변환
```go
package main

import "fmt"

func main() {
    var a int = 42
    var b int64 = int64(a) // int에서 int64로 변환
    fmt.Println("Converted:", b) // 출력: Converted: 42
}
```

### 예제 3: 계산
```go
package main

import "fmt"

func main() {
    var x int64 = 10
    var y int64 = 20
    var product int64 = x * y
    fmt.Println("Product:", product) // 출력: Product: 200
}
```

## 설명
`int64` 타입 사용 시 주의해야 할 점은 다음과 같습니다:

- **오버플로우**: `int64`의 최대값을 초과하는 연산을 수행할 경우 오버플로우가 발생할 수 있습니다. 예를 들어, `9223372036854775807 + 1`을 수행하면 음수로 돌아가는 결과가 발생합니다.
- **타입 변환**: 다른 정수형 타입에서 `int64`로 변환할 때는 명시적으로 변환해 주어야 하며, 변환 시 데이터 손실이 발생하지 않도록 주의해야 합니다.
- **메모리 사용**: `int64`는 8바이트를 차지하므로, 메모리 사용량을 고려하여 필요한 경우 다른 정수형 타입을 사용하는 것이 좋습니다.
