# Go言語における「continue」文の使い方

## 概要
Go言語の「continue」文は、ループ内での処理をスキップし、次のイテレーションに移行するための制御フロー文です。この文を使用することで、特定の条件に基づいて処理を効率的に制御できます。

## ドキュメンテーション
「continue」文は、`for` ループ内で使用され、ループの現在のイテレーションを終了し、次のイテレーションに進むためのものです。これにより、特定の条件が満たされた場合に、ループ内の後続の処理をスキップできます。

### 使い方
「continue」文は次のように使用します。

```go
for i := 0; i < 10; i++ {
    if i%2 == 0 {
        continue // 偶数の場合、次のイテレーションにスキップ
    }
    fmt.Println(i) // 奇数のみを出力
}
```

上記の例では、変数 `i` が偶数の場合に `continue` 文が実行され、`fmt.Println(i)` はスキップされます。これにより、奇数のみが出力されます。

## 例
以下は「continue」文の基本的な使用例です。

### 例1: 偶数をスキップする
```go
package main

import "fmt"

func main() {
    for i := 0; i < 10; i++ {
        if i%2 == 0 {
            continue
        }
        fmt.Println(i) // 出力: 1, 3, 5, 7, 9
    }
}
```

### 例2: 特定の文字をスキップする
```go
package main

import "fmt"

func main() {
    str := "hello world"
    for i, c := range str {
        if c == 'o' {
            continue
        }
        fmt.Printf("Character at %d: %c\n", i, c)
    }
}
```

この例では、文字列内の「o」をスキップして出力します。

## 説明
「continue」文は簡潔で強力な制御文ですが、使用する際にはいくつかの注意点があります。

- **ネストされたループ**: 「continue」文は、現在のループのみに影響を与えます。外側のループには影響を及ぼしません。
- **条件の明確さ**: スキップする条件を明確に設定しないと、意図しない動作を引き起こす可能性があります。条件を慎重に設定することが重要です。

## 一文の要約
