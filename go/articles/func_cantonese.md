# Go 語言中的函數 (func) 使用指南

## 簡介
在 Go 語言中，`func` 是用來定義函數的關鍵字，函數是執行特定任務的代碼塊。函數使得代碼更具可重用性和可讀性，是 Go 語言中重要的組件之一。

## 文檔
### 目的
`func` 關鍵字用於定義函數，這些函數可以接收參數並返回值，允許開發者構建模組化且易於維護的代碼。

### 用法
函數的基本語法如下：
```go
func 函數名稱(參數名稱 參數類型) 返回類型 {
    // 函數體
}
```
- `func`：關鍵字，表示定義函數。
- `函數名稱`：函數的名稱，應該簡潔且具描述性。
- `(參數名稱 參數類型)`：函數的參數列表，可以是一個或多個參數。
- `返回類型`：函數的返回值類型，可以省略，若無返回值則不需要指定。

### 詳細說明
- 函數可以有多個返回值，使用括號將它們包裹起來。
- 可以使用命名返回值，這樣在函數體內直接使用變量名稱即可。
- 函數可以作為一等公民被傳遞和賦值。
- 支援變長參數，使用 `...` 表示可變參數。

## 例子
### 基本用法
以下是一個簡單的函數範例，計算兩個整數的和：
```go
package main

import "fmt"

func add(a int, b int) int {
    return a + b
}

func main() {
    result := add(3, 4)
    fmt.Println("Sum:", result) // 輸出: Sum: 7
}
```

### 具名返回值
以下示例展示了具名返回值的用法：
```go
package main

import "fmt"

func divide(a, b int) (result int, err error) {
    if b == 0 {
        err = fmt.Errorf("除數不能為零")
        return
    }
    result = a / b
    return
}

func main() {
    res, err := divide(10, 2)
    if err != nil {
        fmt.Println(err)
    } else {
        fmt.Println("Result:", res) // 輸出: Result: 5
    }
}
```

## 解釋
在使用 `func` 定義函數時，開發者應注意以下幾點：
- 確保函數名稱具描述性，以便其他開發者能快速理解其功能。
- 避免將過多的邏輯放在單一函數中，應保持函數的單一責任原則。
- 注意錯誤處理，特別是在處理可能出現錯誤的操作時，例如除數為零的情況。

## 一句總結
`func` 是 Go 語言中定義函數的關鍵字，允許開發者以模組化的方式編寫可重用的代碼。
