# Go 語言中的 complex128 型別：深入了解複數

## 簡介
在 Go 語言中，`complex128` 是一個內建的數據類型，用於表示複數，這對於科學計算和工程應用特別重要。它可以有效地處理實數和虛數的運算。

## 文檔
`complex128` 是 Go 語言中表示複數的數據類型，複數由實部和虛部組成，並且可以使用 `real()` 和 `imag()` 函數來提取實部和虛部。`complex128` 型別的實部和虛部都是 `float64` 類型，這意味著它可以表示非常大的數值範圍。

### 用法
在 Go 中，你可以使用 `complex` 函數創建複數，例如：
```go
c := complex(1.0, 2.0) // 1.0 是實部，2.0 是虛部
```
這將創建一個 `complex128` 型別的變數 `c`，其實部為 1.0，虛部為 2.0。

### 詳細說明
- **數學運算**：你可以使用 `+`, `-`, `*`, `/` 等運算符來進行複數的數學運算。
- **內建函數**：
  - `real(c)`：返回複數 `c` 的實部。
  - `imag(c)`：返回複數 `c` 的虛部。
- **類型轉換**：可以將 `complex128` 轉換為 `float64` 以獲取實部或虛部，但不能直接轉換為其他數據類型。

## 範例
以下是一些使用 `complex128` 的基本範例：

```go
package main

import (
    "fmt"
)

func main() {
    // 創建複數
    c1 := complex(1.0, 2.0)
    c2 := complex(3.0, 4.0)

    // 複數加法
    sum := c1 + c2
    fmt.Println("Sum:", sum)

    // 複數實部和虛部
    fmt.Println("Real part:", real(c1))
    fmt.Println("Imaginary part:", imag(c1))
}
```

## 解釋
在使用 `complex128` 時，常見的陷阱包括：
- **運算符優先級**：確保在複雜運算中使用括號，以避免運算順序錯誤。
- **類型不匹配**：在進行運算時，確保參與運算的數據類型正確，否則可能導致編譯錯誤。
- **性能考量**：複數運算相較於實數運算可能稍微慢一些，尤其是在大量計算的情況下，需考慮性能影響。

## 一句總結
`complex128` 是 Go 語言中用於表示和處理複數的數據類型，適合科學計算和工程應用。
