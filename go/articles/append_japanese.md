# Go言語の「append」関数の使い方と詳細ガイド

## 概要
Go言語の「append」関数は、スライスに要素を追加するための組み込み関数です。この関数は、スライスの長さを動的に変更し、新しい要素を追加する際に非常に便利です。

## ドキュメンテーション
### 目的
「append」関数は、既存のスライスに新しい要素を追加するために使用されます。スライスは可変長のデータ構造であり、要素を追加することでそのサイズを変更できます。

### 使用法
「append」関数の基本的な構文は以下の通りです。

```go
func append(slice []Type, elements ...Type) []Type
```

- **slice**: 要素を追加する対象のスライス。
- **elements**: スライスに追加したい要素。

この関数は、新しいスライスを返します。元のスライスは変更されません。

### 詳細
- スライスの容量が十分でない場合、「append」は新しいメモリを割り当て、要素をコピーします。
- 返されたスライスは、元のスライスと同じメモリを指している可能性があるため、元のスライスを変更することに注意が必要です。

## 例
### 基本的な使用法
以下は「append」を使った基本的な例です。

```go
package main

import "fmt"

func main() {
    slice := []int{1, 2, 3}
    slice = append(slice, 4, 5)
    fmt.Println(slice) // 出力: [1 2 3 4 5]
}
```

### スライスの容量が不足している場合
```go
package main

import "fmt"

func main() {
    slice := make([]int, 0, 2) // 容量2のスライスを作成
    slice = append(slice, 1, 2)
    slice = append(slice, 3) // 容量を超えるため、新しいメモリが割り当てられる
    fmt.Println(slice) // 出力: [1 2 3]
}
```

## 説明
「append」関数を使用する際の注意点：
- 複数の要素を追加する場合、可変長引数を利用して一度に追加できます。
- スライスの容量が足りない場合、新しいスライスが作成されるため、元のスライスを保持している場合は新しいスライスを代入する必要があります。
- スライスを直接更新しようとすると、予期しない動作が起こることがあります。

## 一行まとめ
Go言語の「append」関数は、スライスに要素を追加するための便利な組み込み関数で、スライスのサイズを動的に変更します。
