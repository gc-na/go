# Go言語の「for」ループ：基本から応用まで

## 概要
Go言語の「for」ループは、繰り返し処理を行うための基本的な構文です。このループを使用することで、特定の条件が満たされるまでコードブロックを実行することができます。

## ドキュメント
Goの「for」ループは、次の3つの形式で使用できます：

1. **初期化、条件、後処理を持つ形式**：
   ```go
   for 初期化; 条件; 後処理 {
       // 実行するコード
   }
   ```
   - **初期化**：ループが開始される前に一度だけ実行されます。
   - **条件**：ループの各反復前に評価され、条件が真の場合にループが実行されます。
   - **後処理**：各反復の最後に実行されます。

2. **条件のみの形式**：
   ```go
   for 条件 {
       // 実行するコード
   }
   ```
   - この形式では、初期化や後処理を省略できます。条件が真である限り、ループが実行されます。

3. **無限ループの形式**：
   ```go
   for {
       // 実行するコード
   }
   ```
   - 条件を指定しない場合、無限ループが生成されます。ループを終了するには、`break`文を使用します。

## 例
以下は、Go言語の「for」ループの基本的な使用例です。

### 例1: 初期化、条件、後処理を持つループ
```go
package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}
```
この例では、0から4までの数字を出力します。

### 例2: 条件のみのループ
```go
package main

import "fmt"

func main() {
    i := 0
    for i < 5 {
        fmt.Println(i)
        i++
    }
}
```
この例でも、0から4までの数字を出力します。

### 例3: 無限ループ
```go
package main

import "fmt"

func main() {
    i := 0
    for {
        fmt.Println(i)
        i++
        if i >= 5 {
            break
        }
    }
}
```
この例では、無限ループを使用して0から4までの数字を出力します。

## 説明
「for」ループにはいくつかの注意点があります。以下は一般的な落とし穴です。

- **無限ループに注意**：条件を適切に設定しないと、無限ループに陥る危険があります。特に、条件を忘れた無限ループは頻繁に発生します。
- **スコープの理解**：ループ内で宣言された変数は、そのループのスコープ内でのみ有効です。外部からアクセスすることはできません。
- **`continue`と`break`の使用**：`continue`文を使用して現在の反復をスキップし、`break`文を使用してループを完全に終了させることができます。

## 一文要約
Go言語の「for」ループは、初期化、条件、後処理を用いて反復処理を行う強力な構文です。
