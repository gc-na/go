# Go 프로그래밍 언어에서의 "int" 타입

## 개요
Go 프로그래밍 언어에서 "int"는 정수를 표현하기 위한 기본 데이터 타입입니다. 이 타입은 수학적 연산과 다양한 프로그래밍 작업에 널리 사용되며, 플랫폼에 따라 비트 수가 달라질 수 있습니다.

## 문서
### 목적
"int" 타입은 정수 값을 저장하고 조작하기 위해 설계되었습니다. Go에서는 "int"와 "uint" (부호 없는 정수) 타입을 포함하여 여러 정수 타입을 제공합니다. "int"는 일반적으로 가장 많이 사용되며, 시스템 아키텍처에 따라 32비트 또는 64비트로 정의됩니다.

### 사용법
Go에서 "int" 타입은 다음과 같이 선언하고 사용할 수 있습니다:

```go
var num int  // 'num' 변수를 int 타입으로 선언
num = 10     // 'num'에 정수 10 할당
```

Go에서는 정수 리터럴을 사용할 때 특별한 표기법이 필요하지 않으며, 기본적으로 "int" 타입으로 간주됩니다.

### 세부 사항
- "int"는 기본적으로 부호 있는 정수 타입입니다.
- "int"의 크기는 시스템에 따라 달라질 수 있으며, 32비트 시스템에서는 32비트 정수, 64비트 시스템에서는 64비트 정수를 나타냅니다.
- "int" 타입은 산술 연산, 비트 연산 등 다양한 연산에 사용할 수 있습니다.
- Go는 타입 안전성을 보장하므로, 다른 타입과의 직접적인 연산은 허용되지 않으며, 변환이 필요합니다.

## 예제
다음은 "int" 타입의 기본 사용 예제입니다:

```go
package main

import "fmt"

func main() {
    var a int = 5
    var b int = 10
    sum := a + b        // 덧셈
    fmt.Println(sum)    // 결과: 15

    var c int = -3
    product := a * c    // 곱셈
    fmt.Println(product) // 결과: -15
}
```

## 설명
- **일반적인 함정**: "int" 타입은 플랫폼에 따라 크기가 다를 수 있으므로, 이식성에 주의해야 합니다. 예를 들어, 32비트와 64비트 시스템에서의 메모리 사용량이 다를 수 있습니다.
- **타입 변환**: 다른 타입과의 연산을 수행할 때는 반드시 명시적인 타입 변환이 필요합니다. 예를 들어, "int"와 "float64"를 함께 사용하려면 "float64(num)"와 같이 변환해야 합니다.
