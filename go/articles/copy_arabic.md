# النسخ في لغة Go: كيفية استخدام دالة copy 

## الملخص
تُستخدم دالة `copy` في لغة Go لنسخ عناصر من شريحة (slice) إلى أخرى، مما يسهل إدارة البيانات والتلاعب بها في الذاكرة.

## الوثائق
### الغرض
تعمل دالة `copy` على نسخ العناصر من شريحة المصدر إلى شريحة الوجهة. إذا كانت الشريحتان مختلفتين في الحجم، فسيتم نسخ العدد الأقل من العناصر بين الشريحتين.

### الاستخدام
تُستخدم `copy` بالشكل التالي:
```go
copy(dst []Type, src []Type) int
```
- `dst`: شريحة الوجهة التي ستُنسخ إليها البيانات.
- `src`: شريحة المصدر التي سيتم نسخ البيانات منها.
- تُرجع الدالة عدد العناصر المنسوخة.

### التفاصيل
- لا يتطلب أن تكون الشريحتان من نفس الحجم، لكن يُفضل أن تكون شريحة الوجهة كبيرة بما يكفي لاستيعاب العناصر المنسوخة.
- إذا كانت الشريحة الوجهة أصغر من شريحة المصدر، فسيتم نسخ فقط عدد العناصر المناسبة التي يمكن استيعابها.
- في حالة كانت الشريحة الوجهة أكبر، فستبقى العناصر الزائدة في الشريحة دون تغيير.

## الأمثلة
### مثال 1: نسخ شريحة بسيطة
```go
package main

import "fmt"

func main() {
    src := []int{1, 2, 3, 4, 5}
    dst := make([]int, 3)
    n := copy(dst, src)
    fmt.Println(dst) // Output: [1 2 3]
    fmt.Println(n)   // Output: 3
}
```

### مثال 2: نسخ شريحة أكبر من المصدر
```go
package main

import "fmt"

func main() {
    src := []string{"a", "b", "c"}
    dst := make([]string, 5)
    n := copy(dst, src)
    fmt.Println(dst) // Output: [a b c ]
    fmt.Println(n)   // Output: 3
}
```

### مثال 3: نسخ شريحة أصغر
```go
package main

import "fmt"

func main() {
    src := []int{10, 20, 30, 40}
    dst := make([]int, 2)
    n := copy(dst, src)
    fmt.Println(dst) // Output: [10 20]
    fmt.Println(n)   // Output: 2
}
```

## الشرح
### الأخطاء الشائعة
- **عدم كفاية حجم شريحة الوجهة**: إذا كانت شريحة الوجهة أصغر من شريحة المصدر، ستفقد بعض العناصر.
- **عدم التحقق من القيم بعد النسخ**: قد يحدث ارتباك حول ما تم نسخه، لذا من الأفضل دائمًا فحص عدد العناصر المنسوخة.

### ملاحظات إضافية
- الدالة `copy` تعمل على نسخ العناصر عن طريق القيمة، وليست عن طريق المرجع، مما يعني أن التغييرات على العناصر المنسوخة في شريحة الوجهة لن تؤثر على شريحة المصدر.

## ملخص جملة واحدة
تُستخدم دالة `copy` في Go لنسخ العناصر من شريحة إلى أخرى، مما يسهل إدارة البيانات بشكل فعال.
