# استيراد الحزم في لغة Go: دليل شامل

## ملخص
يتيح الأمر "import" في لغة Go للمطورين إمكانية استيراد الحزم (Packages) واستخدامها في مشاريعهم، مما يسهل إدارة الاعتماديات ويعزز قابلية إعادة استخدام الكود.

## الوثائق
يُستخدم الأمر "import" في Go لتعريف الحزم التي سيتم استخدامها في ملف المصدر. الحزم هي مجموعات من التعليمات البرمجية التي يمكن أن تحتوي على أنواع بيانات، دوال، واجهات، إلخ. يسمح "import" للمطورين بالاستفادة من الكود المكتوب مسبقًا من قبل الآخرين أو من كودهم الخاص.

### الغرض
الغرض الأساسي من استخدام الأمر "import" هو تمكين الوصول إلى الوظائف والأدوات الموجودة في حزم أخرى، مما يسهل تطوير التطبيقات المعقدة.

### الاستخدام
يتم استخدام "import" في بداية ملف المصدر، حيث يمكن استيراد حزمة واحدة أو أكثر. يمكن كتابة الأمر كالتالي:

```go
import "اسم_الحزمة"
```

كما يمكن استيراد عدة حزم باستخدام الأقواس:

```go
import (
    "حزمة_1"
    "حزمة_2"
)
```

## أمثلة
### مثال 1: استيراد حزمة واحدة
```go
package main

import "fmt"

func main() {
    fmt.Println("مرحباً بالعالم!")
}
```

### مثال 2: استيراد عدة حزم
```go
package main

import (
    "fmt"
    "math"
)

func main() {
    fmt.Println("مربع العدد 4 هو:", math.Pow(4, 2))
}
```

## شرح
### الأخطاء الشائعة
- **عدم وجود الحزمة**: عند محاولة استيراد حزمة غير موجودة، ستظهر رسالة خطأ تفيد بأن الحزمة غير متاحة.
- **أسماء الحزم المتشابهة**: إذا كانت هناك حزم تحمل أسماء مشابهة، يجب استخدام الأسماء الكاملة لتجنب الالتباس.
- **عدم استخدام الحزمة**: إذا تم استيراد حزمة ولكن لم يتم استخدامها، سيظهر تحذير من المترجم.

### ملاحظات إضافية
- تستخدم Go نظام الحزم بشكل متسق، مما يجعل من السهل على المطورين تنظيم الكود.
- يمكن استيراد حزم محلية (تم إنشاؤها داخل المشروع) أو حزم من الإنترنت (مثل حزم من GitHub).

## ملخص من سطر واحد
يتيح الأمر "import" في لغة Go استيراد الحزم واستخدامها لتسهيل تطوير التطبيقات وتعزيز إعادة استخدام الكود.
