# جملة "if" في لغة البرمجة Go: الاستخدامات والتفاصيل

## الملخص
تعتبر جملة "if" واحدة من العناصر الأساسية في لغة البرمجة Go، حيث تُستخدم لتحديد تدفق البرنامج بناءً على شروط معينة.

## الوثائق
جملة "if" في Go تُستخدم لتنفيذ كود معين فقط إذا كانت شرط معين صحيح. يمكن استخدام جملة "if" مع شرط بسيط أو مع شروط مركبة. تُعتبر "if" من الأدوات الأساسية للتحكم في تدفق البرنامج.

### الاستخدام
تتبع جملة "if" التركيب التالي:

```go
if الشرط {
    // كود يتم تنفيذه إذا كان الشرط صحيحًا
}
```

### تفاصيل
- يمكن استخدام جملة "if" مع جملة "else" لتحديد حالة بديلة عندما يكون الشرط غير صحيح.
- يمكن أيضًا استخدام "else if" لتحديد شروط إضافية.
- يُسمح باستخدام الأقواس لتجميع عدة شروط معًا.

## أمثلة
### مثال 1: جملة if بسيطة
```go
package main

import "fmt"

func main() {
    x := 10
    if x > 5 {
        fmt.Println("x أكبر من 5")
    }
}
```

### مثال 2: استخدام else
```go
package main

import "fmt"

func main() {
    x := 3
    if x > 5 {
        fmt.Println("x أكبر من 5")
    } else {
        fmt.Println("x ليس أكبر من 5")
    }
}
```

### مثال 3: استخدام else if
```go
package main

import "fmt"

func main() {
    x := 5
    if x > 5 {
        fmt.Println("x أكبر من 5")
    } else if x == 5 {
        fmt.Println("x يساوي 5")
    } else {
        fmt.Println("x أقل من 5")
    }
}
```

## الشرح
- **المزالق الشائعة**: تأكد من أن الشرط داخل جملة "if" يُقيّم بشكل صحيح. قد تحدث مشاكل إذا تم استخدام المتغيرات غير المعينة أو القيم غير المتوقعة.
- **ملاحظات إضافية**: في Go، يمكنك أيضًا استخدام تعبيرات متعددة في جملة "if"، مما يتيح لك تعريف متغيرات جديدة في نفس السطر.

## ملخص جملة واحدة
تعتبر جملة "if" في Go أداة أساسية للتحكم في تدفق البرنامج بناءً على شروط محددة.
