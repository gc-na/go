# Go의 "default" 키워드: 기본값과 사용법

## 개요
Go 프로그래밍 언어에서 "default"는 switch 문에서 특정 조건이 충족되지 않을 때 실행되는 기본 동작을 정의하는 데 사용됩니다. 이 기능은 코드의 가독성과 유지보수성을 높이는 데 기여합니다.

## 문서화
"Default" 키워드는 Go의 제어 구조 중 하나인 switch 문에서 사용됩니다. switch 문은 여러 조건을 평가하여 해당 조건이 충족될 경우 특정 코드를 실행합니다. 만약 모든 조건이 충족되지 않는 경우, default 절을 사용하여 실행할 코드를 지정할 수 있습니다. 이는 조건이 모두 일치하지 않을 때의 기본 동작을 정의하는 데 유용합니다.

### 사용법
기본적인 switch 문은 다음과 같은 형식으로 사용됩니다:

```go
switch 변수 {
case 값1:
    // 값1이 변수와 일치할 경우 실행되는 코드
case 값2:
    // 값2가 변수와 일치할 경우 실행되는 코드
default:
    // 어떤 값도 일치하지 않을 경우 실행되는 코드
}
```

위의 예에서 변수의 값이 값1이나 값2와 일치하지 않을 경우 default 절의 코드가 실행됩니다.

## 예제
아래는 "default" 키워드를 사용하는 간단한 예제입니다:

```go
package main

import (
    "fmt"
)

func main() {
    day := "월요일"

    switch day {
    case "토요일":
        fmt.Println("주말입니다!")
    case "일요일":
        fmt.Println("주말입니다!")
    default:
        fmt.Println("주중입니다.")
    }
}
```

위 코드는 변수 day의 값이 "토요일"이나 "일요일"일 경우 주말을 출력하고, 그 외의 경우에는 주중이라는 메시지를 출력합니다.

## 설명
"Default" 절을 사용할 때 주의해야 할 점은 switch 문의 조건 순서입니다. 조건이 매치되면 해당 조건의 코드 블록이 실행되며, 이후의 조건은 평가되지 않습니다. 따라서 default 절은 마지막에 위치해야 합니다. 

또한, default 절은 선택적이므로 필요에 따라 생략할 수 있습니다. 하지만 명확한 코드 작성을 위해 default 절을 사용하는 것이 좋습니다. 

## 한 줄 요약
