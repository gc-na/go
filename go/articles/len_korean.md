# Go의 len 함수: 슬라이스, 배열, 문자열의 길이 측정하기

## 개요
Go 프로그래밍 언어에서 `len` 함수는 슬라이스, 배열, 문자열, 맵 등의 길이를 반환하는 내장 함수입니다. 이 함수는 데이터 구조의 크기를 파악하는 데 유용하며, 간단한 문법으로 사용될 수 있습니다.

## 문서화
### 목적
`len` 함수는 다양한 데이터 타입의 길이를 측정하여, 프로그래밍 과정에서 데이터의 크기를 확인하고 조건문 등을 통해 로직을 제어하는 데 중요한 역할을 합니다.

### 사용법
`len` 함수는 다음과 같은 형식으로 사용됩니다:

```go
len(v interface{}) int
```

여기서 `v`는 길이를 측정하고자 하는 데이터 구조입니다. 반환값은 `int` 타입으로, 요청된 데이터 구조의 요소 개수나 문자 수를 나타냅니다.

### 지원되는 데이터 타입
- **슬라이스**: 요소 개수 반환
- **배열**: 요소 개수 반환
- **문자열**: 문자 수 반환 (바이트 수를 기준으로 계산)
- **맵**: 키-값 쌍의 개수 반환
- **채널**: 버퍼의 크기 반환

## 예제
다음은 `len` 함수를 사용하는 몇 가지 기본적인 예제입니다.

### 슬라이스의 길이
```go
package main

import "fmt"

func main() {
    slice := []int{1, 2, 3, 4, 5}
    fmt.Println("슬라이스의 길이:", len(slice)) // 출력: 슬라이스의 길이: 5
}
```

### 배열의 길이
```go
package main

import "fmt"

func main() {
    array := [3]int{1, 2, 3}
    fmt.Println("배열의 길이:", len(array)) // 출력: 배열의 길이: 3
}
```

### 문자열의 길이
```go
package main

import "fmt"

func main() {
    str := "안녕하세요"
    fmt.Println("문자열의 길이:", len(str)) // 출력: 문자열의 길이: 15 (UTF-8 바이트 수)
}
```

### 맵의 길이
```go
package main

import "fmt"

func main() {
    m := map[string]int{"a": 1, "b": 2}
    fmt.Println("맵의 길이:", len(m)) // 출력: 맵의 길이: 2
}
```

## 설명
- `len` 함수는 매우 직관적이지만, 사용 시 주의해야 할 점이 있습니다. 문자열의 길이는 바이트 수로 반환되므로, UTF-8 인코딩의 다국어 문자열에서는 문자 수와 다를 수 있습니다. 예를 들어, 한글과 같은 다국어 문자는 여러 바이트로 표현될 수 있습니다.
- 슬라이스나 배열의 경우, 길이는 그 자체의 크기와 같지만, 용량(capacity)과는 다른 점을 기억해야 합니다. 용량은 슬라이스의 최대 크기를 뜻하는 반면, 길이는 현재 포함된 요소의 수를 나타냅니다.

## 한 줄 요약
Go의 `len` 함수는 슬라이스, 배열, 문자열, 맵 등의 길이를 반환하여 데이터 구조의 크기를 쉽게 확인할 수 있는 유용한 도구입니다.
