# Go의 변수 선언: `var` 키워드에 대한 완벽 가이드

## 개요
Go 언어에서 `var` 키워드는 변수를 선언하는 데 사용되는 기본적인 명령어로, 타입에 따라 다양한 데이터 유형의 변수를 생성할 수 있습니다. 이 문서에서는 `var` 키워드의 목적, 사용법 및 여러 예제를 통해 Go 언어에서의 변수 선언을 상세히 설명합니다.

## 문서화
`var` 키워드는 Go에서 변수를 선언할 때 사용하는 키워드입니다. 변수를 선언하면 메모리에 공간이 할당되며, 이후 프로그램에서 해당 변수를 사용하여 데이터를 저장하고 조작할 수 있습니다.

### 목적
- `var`를 사용하여 변수를 선언함으로써 데이터의 저장 공간을 할당합니다.
- 변수를 선언할 때 데이터 타입을 명시하여, 컴파일러가 해당 변수의 데이터 유형을 인식하고 타입 안전성을 보장합니다.

### 사용법
변수 선언은 다음과 같은 형식으로 이루어집니다:

```go
var variableName type
```

여기서 `variableName`은 변수의 이름, `type`은 변수의 데이터 유형입니다. 여러 변수를 한 번에 선언할 수도 있습니다:

```go
var var1, var2 type1
var3, var4 type2
```

또한, 변수를 선언하면서 초기값을 부여할 수도 있습니다:

```go
var variableName type = initialValue
```

## 예제
1. 기본 변수 선언:

```go
var age int
age = 30
```

2. 여러 변수 선언:

```go
var x, y int
x = 10
y = 20
```

3. 초기값을 가진 변수 선언:

```go
var name string = "Alice"
```

4. 타입 추론을 통한 선언:

```go
var score = 95.5  // float64 타입으로 추론됨
```

5. 블록 내에서 변수 선언:

```go
func main() {
    var message string = "Hello, World!"
    println(message)
}
```

## 설명
`var` 키워드를 사용할 때 주의해야 할 점은 다음과 같습니다:

- 변수 이름은 반드시 유효한 식별자여야 하며, 대소문자를 구분합니다.
- Go에서는 기본적으로 변수를 선언할 때 초기값을 주지 않으면 해당 데이터 타입의 제로값(예: int는 0, string은 "")으로 초기화됩니다.
- 동일한 범위 내에서 같은 이름의 변수를 중복 선언할 수 없으며, 이는 컴파일 오류를 발생시킵니다.
- Go에서는 변수를 선언한 후 사용하지 않으면 컴파일러가 경고를 발생시킵니다. 이를 방지하기 위해 변수를 선언한 후 반드시 활용해야 합니다.

## 한 문장 요약
Go 언어에서 `var` 키워드는 다양한 데이터 유형의 변수를 선언하고 초기화하는 데 사용되는 기본 키워드입니다.
