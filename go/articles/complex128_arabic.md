# النوع complex128 في لغة Go: كل ما تحتاج معرفته

## ملخص
يعتبر النوع `complex128` في لغة Go نوع بيانات يمثل الأعداد المركبة بدقة عالية. يتكون هذا النوع من جزئين: الجزء الحقيقي والجزء التخيلي، كل منهما يمثل عددًا من نوع `float64`.

## الوثائق
### الغرض
يستخدم النوع `complex128` لتخزين الأعداد المركبة، وهي أعداد تتكون من جزء حقيقي وآخر تخيلي. يمكن استخدام هذا النوع في العمليات الرياضية المعقدة، مثل تلك المستخدمة في الهندسة، الفيزياء، والبرمجة العلمية.

### الاستخدام
لإنشاء عدد مركب من نوع `complex128`، يمكنك استخدام الصيغة التالية:

```go
z := complex(real, imag)
```

حيث `real` هو الجزء الحقيقي و`imag` هو الجزء التخيلي.

### التفاصيل
- **النوع**: `complex128`
- **الحجم**: 16 بايت
- **المكونات**: 
  - الجزء الحقيقي: `float64`
  - الجزء التخيلي: `float64`

يمكنك استخدام العمليات الرياضية الأساسية مثل الجمع، الطرح، الضرب، والقسمة مع الأعداد المركبة.

## الأمثلة
### مثال 1: إنشاء عدد مركب
```go
package main

import (
    "fmt"
)

func main() {
    z1 := complex(1.0, 2.0) // 1 + 2i
    fmt.Println(z1)
}
```

### مثال 2: جمع عددين مركبين
```go
package main

import (
    "fmt"
)

func main() {
    z1 := complex(1.0, 2.0) // 1 + 2i
    z2 := complex(3.0, 4.0) // 3 + 4i
    z3 := z1 + z2          // (1 + 2i) + (3 + 4i) = 4 + 6i
    fmt.Println(z3)
}
```

### مثال 3: ضرب عددين مركبين
```go
package main

import (
    "fmt"
)

func main() {
    z1 := complex(1.0, 2.0) // 1 + 2i
    z2 := complex(3.0, 4.0) // 3 + 4i
    z3 := z1 * z2          // (1 + 2i) * (3 + 4i) = -5 + 10i
    fmt.Println(z3)
}
```

## الشرح
### الأخطاء الشائعة
- **عدم استخدام الوحدات الصحيحة**: تأكد من أنك تستخدم `float64` كجزء حقيقي وتخيلي عند إنشاء الأعداد المركبة.
- **تجاهل دقة الأعداد**: عند التعامل مع العمليات المركبة، تأكد من أنك تأخذ في الاعتبار دقة الأعداد، حيث قد تؤدي الأخطاء التراكمية إلى نتائج غير دقيقة.

### ملاحظات إضافية
يمكنك استخدام الدوال المدمجة مثل `real()` و `imag()` لاستخراج الأجزاء الحقيقية والتخييلية من العدد المركب.

## ملخص جملة واحدة
`complex128` هو نوع بيانات في لغة Go يمثل الأعداد المركبة بدقة عالية، ويتضمن جزءًا حقيقيًا وآخر تخيليًا، مما يسهل إجراء العمليات الرياضية المعقدة.
