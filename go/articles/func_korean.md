# Go 언어에서의 "func": 함수 정의 및 사용법

## 개요
Go 언어에서 "func" 키워드는 함수를 정의하고 호출하는 데 사용됩니다. 함수는 특정 작업을 수행하는 코드 블록으로, 재사용성과 모듈화를 통해 코드의 가독성과 유지보수성을 향상시킵니다.

## 문서화
"func" 키워드는 Go 언어에서 함수의 선언을 시작하는 데 사용됩니다. 함수는 입력 인자를 받고, 특정 작업을 수행한 후 결과를 반환할 수 있습니다. Go에서는 함수를 변수처럼 다룰 수 있으며, 고차 함수와 클로저와 같은 기능도 지원합니다.

### 정의
함수는 다음과 같은 형식으로 정의됩니다:

```go
func 함수명(매개변수1 타입1, 매개변수2 타입2) 반환타입 {
    // 함수 본체
}
```

### 사용법
1. **함수 선언**: 함수는 `func` 키워드로 시작하며, 함수 이름, 매개변수 리스트, 반환 타입, 그리고 함수 본체를 포함합니다.
2. **함수 호출**: 함수를 호출할 때는 함수 이름 뒤에 괄호를 붙이고, 필요한 경우 매개변수를 전달합니다.

### 예제
아래는 Go에서 함수의 기본 사용법을 보여주는 간단한 예제입니다.

```go
package main

import "fmt"

// 두 수를 더하는 함수
func add(a int, b int) int {
    return a + b
}

func main() {
    result := add(3, 5)
    fmt.Println("Result:", result) // 출력: Result: 8
}
```

## 설명
### 일반적인 오류 및 주의사항
- **매개변수 생략**: 모든 매개변수는 함수 정의에서 명시해야 하며, 생략할 경우 컴파일 오류가 발생합니다.
- **반환값 처리**: 함수가 반환하는 값을 사용하지 않으면 경고가 발생할 수 있습니다. 필요하지 않을 경우 `_`을 사용하여 무시할 수 있습니다.
- **변수 스코프**: 함수 내에서 선언된 변수는 해당 함수 내에서만 유효합니다. 이 점을 유의하여 변수를 사용해야 합니다.

## 한 줄 요약
Go 언어에서 "func"는 함수 정의 및 호출을 위한 핵심 키워드로, 코드의 재사용성과 모듈화를 가능하게 합니다.
