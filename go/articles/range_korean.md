# Go 언어에서의 range: 배열, 슬라이스, 맵을 순회하는 방법

## 개요
Go 언어에서 `range` 키워드는 배열, 슬라이스, 맵, 채널 등 다양한 자료구조를 순회할 수 있는 강력한 기능을 제공합니다. 이 기능은 반복문과 함께 사용되어 코드의 가독성을 높이고, 반복 작업을 간편하게 수행할 수 있게 해줍니다.

## 문서화
`range`는 Go의 `for` 루프와 함께 사용되어, 다양한 자료구조의 요소를 순회하는 데 사용됩니다. 배열, 슬라이스, 맵, 채널에 대해 각각의 요소를 쉽게 접근할 수 있는 방법을 제공합니다.

### 사용 목적
- 배열이나 슬라이스의 요소를 반복적으로 처리하기 위해.
- 맵의 키-값 쌍을 순회하기 위해.
- 채널의 데이터를 수신하기 위해.

### 사용법
`range`는 다음과 같은 구문으로 사용됩니다:

```go
for index, value := range collection {
    // 처리할 코드
}
```

- `index`는 현재 반복의 인덱스를 나타내며, `value`는 해당 인덱스의 값을 나타냅니다.
- 컬렉션은 배열, 슬라이스, 맵, 또는 채널이 될 수 있습니다.

## 예제
### 배열과 슬라이스 예제
```go
numbers := []int{1, 2, 3, 4, 5}
for index, value := range numbers {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}
```

### 맵 예제
```go
person := map[string]int{"Alice": 25, "Bob": 30}
for key, value := range person {
    fmt.Printf("Key: %s, Value: %d\n", key, value)
}
```

### 채널 예제
```go
ch := make(chan int)
go func() {
    for i := 0; i < 5; i++ {
        ch <- i
    }
    close(ch)
}()
for value := range ch {
    fmt.Println(value)
}
```

## 설명
`range`를 사용할 때 주의해야 할 몇 가지 점이 있습니다:

1. **인덱스 생략**: 필요한 경우 `range`에서 인덱스를 생략할 수 있습니다. 예를 들어, 슬라이스를 순회할 때 값만 필요하다면 다음과 같이 작성할 수 있습니다.
   ```go
   for _, value := range numbers {
       fmt.Println(value)
   }
   ```

2. **맵의 순회 순서**: 맵은 순서가 없는 자료구조이므로, `range`로 순회할 때마다 순서가 달라질 수 있습니다.

3. **채널 수신**: 채널에서 `range`를 사용하여 데이터를 수신할 때는 채널을 닫아야 순회가 종료됩니다. 그렇지 않으면 무한 루프가 발생할 수 있습니다.

4. **복사본**: `range`로 순회할 때, 값이 아닌 포인터를 사용하는 경우 주의해야 합니다. 값의 복사본이 생성되기 때문에 예상치 못한 결과를 초래할 수 있습니다.

## 한줄 요약
Go 언어의 `range` 키워드는 배열, 슬라이스, 맵, 채널 등을 간편하게 순회할 수 있는 기능을 제공합니다.
