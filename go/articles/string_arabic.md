# السلاسل النصية في لغة Go: دليل شامل

## ملخص
تعتبر السلاسل النصية (strings) في لغة Go نوعاً أساسياً يُستخدم لتمثيل النصوص والبيانات النصية، حيث تقدم مجموعة من الوظائف التي تسهل التعامل مع البيانات النصية بكفاءة وسهولة.

## الوثائق
تستخدم السلاسل النصية في Go لتخزين النصوص، وهي نوع ثابت الطول عند تعريفها. تُعتبر السلاسل النصية في Go غير قابلة للتغيير، مما يعني أنه بمجرد إنشاء سلسلة نصية، لا يمكن تعديلها. يُمكن استخدام الحزمة المدمجة `strings` لتوفير مجموعة من الدوال المفيدة للتعامل مع السلاسل النصية.

### الاستخدام
لإنشاء سلسلة نصية في Go، يمكنك استخدام الأقواس المزدوجة:

```go
var str string = "مرحبا بالعالم"
```

يمكنك أيضاً استخدام الدالة `fmt.Printf` لطباعة السلاسل النصية:

```go
fmt.Printf("النص هو: %s\n", str)
```

### الوظائف الأساسية
- **طول السلسلة**: يمكنك استخدام الدالة `len()` للحصول على طول السلسلة النصية.
- **التحويل إلى حرف كبير**: يمكنك استخدام `strings.ToUpper()`.
- **التحويل إلى حرف صغير**: يمكنك استخدام `strings.ToLower()`.
- **البحث عن النص**: يمكنك استخدام `strings.Contains()` للبحث عن نص معين داخل سلسلة نصية.

## أمثلة
### إنشاء سلسلة نصية
```go
package main

import "fmt"

func main() {
    greeting := "مرحبا بالعالم"
    fmt.Println(greeting)
}
```

### استخدام دوال السلسلة
```go
package main

import (
    "fmt"
    "strings"
)

func main() {
    str := "مرحبا بالعالم"
    fmt.Println("طول السلسلة:", len(str))
    fmt.Println("السلسلة بحروف كبيرة:", strings.ToUpper(str))
    fmt.Println("السلسلة بحروف صغيرة:", strings.ToLower(str))
    fmt.Println("هل تحتوي السلسلة على 'العالم'؟", strings.Contains(str, "العالم"))
}
```

## الشرح
### الأخطاء الشائعة
- **عدم فهم السلاسل غير القابلة للتغيير**: قد يعتقد المطورون الجدد أنه يمكن تعديل السلاسل النصية بعد إنشائها، ولكن يجب تذكيرهم بأنها غير قابلة للتغيير. إذا كنت بحاجة إلى تعديل نص، يجب عليك إنشاء سلسلة جديدة.
- **الخلط بين `string` و `[]byte`**: السلاسل النصية في Go هي نوع منفصل عن مصفوفات البايت. إذا كنت بحاجة إلى تحويل سلسلة نصية إلى مصفوفة بايت، يمكنك استخدام `[]byte(str)`.

### ملاحظات إضافية
تعتبر السلاسل النصية في Go فعالة من حيث الأداء وتدعم معالجة النصوص بشكل سلس. من المهم استكشاف الحزمة `strings` للحصول على أقصى استفادة من الوظائف المتاحة.

## ملخص جملة واحدة
تعتبر السلاسل النصية في لغة Go نوعاً أساسياً للتعامل مع البيانات النصية، وتقدم مجموعة متنوعة من الوظائف المفيدة.
