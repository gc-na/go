# استخدام جملة "else" في لغة Go: دليل شامل

## ملخص
تُستخدم جملة "else" في لغة Go كجزء من التحكم في التدفق، حيث تتيح تنفيذ كود معين إذا لم تتحقق شرطية جملة "if" السابقة.

## الوثائق
تُعتبر جملة "else" واحدة من أبرز أدوات التحكم في التدفق في لغة Go. تُستخدم مع جملة "if" لتحديد الكود الذي يجب تنفيذه عندما تكون الشرطية غير صحيحة. يمكن استخدام "else" بمفردها أو بالتزامن مع "if" و"else if" لتشكيل سلسلة من الشروط.

### الاستخدام
تأتي جملة "else" بعد جملة "if" وتستخدم لتحديد الكود البديل الذي سيُنفذ في حالة عدم تحقق شرط "if". الشكل الأساسي لاستخدام "else" هو كما يلي:

```go
if الشرط {
    // تنفيذ هذا الكود إذا كان الشرط صحيحًا
} else {
    // تنفيذ هذا الكود إذا كان الشرط غير صحيح
}
```

### التفاصيل
- **التداخل**: يمكن استخدام "else" مع "if" و"else if" لعمل سلسلة من الشروط.
- **الشرطيات**: يمكن أن تحتوي الشرطيات على تعبيرات منطقية معقدة.
- **التنسيق**: يجب أن تكون هناك مسافة فاصلة (space) بين "if" و"else".

## أمثلة
### مثال 1: استخدام جملة "else"
```go
package main

import "fmt"

func main() {
    number := 10

    if number%2 == 0 {
        fmt.Println("العدد زوجي")
    } else {
        fmt.Println("العدد فردي")
    }
}
```

### مثال 2: استخدام "else if"
```go
package main

import "fmt"

func main() {
    score := 85

    if score >= 90 {
        fmt.Println("ممتاز")
    } else if score >= 75 {
        fmt.Println("جيد")
    } else {
        fmt.Println("مقبول")
    }
}
```

## الشرح
عند استخدام جملة "else"، يجب الانتباه إلى النقاط التالية:
- **عدم وجود شرط**: لا تحتاج جملة "else" إلى شرط، فهي تُنفذ دائمًا إذا لم يتحقق الشرط السابق.
- **تداخل الشروط**: يمكن أن يؤدي الاستخدام الخاطئ لتداخل الشروط إلى نتائج غير متوقعة، لذا يجب تنظيم الكود بشكل جيد.
- **الأداء**: ليس هناك تأثير كبير على الأداء عند استخدام "else"، ولكن يجب تجنب التعقيد غير الضروري في الشروط.

## ملخص جملة واحدة
تُستخدم جملة "else" في لغة Go لتوفير مسار بديل للتنفيذ عندما تكون الشرطية في جملة "if" غير صحيحة.
